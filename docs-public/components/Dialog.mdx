# Dialog

Modal overlays for focused interactions and important messages. Manages focus, prevents background interaction, and provides smooth animations.

## Import

```tsx
import { Dialog } from "liquidify-react/dialog"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<Dialog.Root>
			<Dialog.Trigger asChild>
				<Button>Open Dialog</Button>
			</Dialog.Trigger>
			
			<Dialog.Backdrop />
			<Dialog.Positioner>
				<Dialog.Content>
					<Dialog.Title>Confirm Action</Dialog.Title>
					<Dialog.Description>
						Are you sure you want to delete this item? This action cannot be undone.
					</Dialog.Description>
					
					<div style={{ display: "flex", gap: "1rem", marginTop: "1.5rem" }}>
						<Dialog.CloseTrigger asChild>
							<Button variant="secondary">Cancel</Button>
						</Dialog.CloseTrigger>
						<Button variant="destructive">Delete</Button>
					</div>
					
					<Dialog.CloseTrigger asChild>
						<button style={{ position: "absolute", top: "1rem", right: "1rem" }}>
							<XIcon />
						</button>
					</Dialog.CloseTrigger>
				</Dialog.Content>
			</Dialog.Positioner>
		</Dialog.Root>
	)
}
```

## Controlled State

```tsx
import { useState } from "react"

export default function ControlledExample() {
	const [open, setOpen] = useState(false)
	
	return (
		<div>
			<p>Dialog is {open ? "open" : "closed"}</p>
			<Dialog.Root open={open} onOpenChange={setOpen}>
				<Dialog.Trigger asChild>
					<Button>Show Settings</Button>
				</Dialog.Trigger>
				
				<Dialog.Backdrop />
				<Dialog.Positioner>
					<Dialog.Content>
						<Dialog.Title>Settings</Dialog.Title>
						<Dialog.Description>
							Configure your application preferences.
						</Dialog.Description>
						
						<form onSubmit={(e) => { e.preventDefault(); setOpen(false) }}>
							<div style={{ margin: "1rem 0" }}>
								<label>
									<input type="checkbox" /> Enable notifications
								</label>
							</div>
							<div style={{ margin: "1rem 0" }}>
								<label>
									<input type="checkbox" /> Dark mode
								</label>
							</div>
							
							<div style={{ display: "flex", gap: "1rem", marginTop: "1.5rem" }}>
								<Button type="submit">Save</Button>
								<Dialog.CloseTrigger asChild>
									<Button variant="secondary">Cancel</Button>
								</Dialog.CloseTrigger>
							</div>
						</form>
					</Dialog.Content>
				</Dialog.Positioner>
			</Dialog.Root>
		</div>
	)
}
```

## Alert Dialog

```tsx
export default function AlertExample() {
	return (
		<Dialog.Root role="alertdialog">
			<Dialog.Trigger asChild>
				<Button variant="destructive">Delete Account</Button>
			</Dialog.Trigger>
			
			<Dialog.Backdrop />
			<Dialog.Positioner>
				<Dialog.Content>
					<Dialog.Title>Delete Account</Dialog.Title>
					<Dialog.Description>
						This will permanently delete your account and all associated data. 
						This action cannot be undone.
					</Dialog.Description>
					
					<div style={{ display: "flex", gap: "1rem", marginTop: "1.5rem" }}>
						<Dialog.CloseTrigger asChild>
							<Button variant="secondary">Keep Account</Button>
						</Dialog.CloseTrigger>
						<Button variant="destructive">
							Yes, Delete Account
						</Button>
					</div>
				</Dialog.Content>
			</Dialog.Positioner>
		</Dialog.Root>
	)
}
```

## API Reference

### Root Props
- `open`: `boolean` - Controlled open state
- `defaultOpen`: `boolean` - Default open state
- `onOpenChange`: `(open: boolean) => void` - State change handler
- `modal`: `boolean` - Modal behavior (default: true)
- `role`: `"dialog" | "alertdialog"` - ARIA role

### Accessibility
- Focus trap within dialog content
- Escape key closes dialog
- Click outside closes dialog (configurable)
- Proper ARIA attributes and roles
- Screen reader announcements
