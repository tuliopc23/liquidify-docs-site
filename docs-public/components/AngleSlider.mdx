# AngleSlider

Circular slider for selecting angle values. Ideal for rotation controls, color wheels, compass directions, and any interface requiring precise angular input.

## Import

```tsx
import { AngleSlider } from "liquidify-react/angle-slider"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<AngleSlider.Root defaultValue={45}>
			<AngleSlider.Label>Rotation</AngleSlider.Label>
			<AngleSlider.Control>
				<AngleSlider.Track />
				<AngleSlider.Range />
				<AngleSlider.Thumb />
			</AngleSlider.Control>
			<AngleSlider.ValueText />
		</AngleSlider.Root>
	)
}
```

## With Custom Range

```tsx
export default function CustomRangeExample() {
	return (
		<AngleSlider.Root min={0} max={180} defaultValue={90}>

```

## API Reference

### Root Props
- `value`: `number` - Current angle value
- `defaultValue`: `number` - Default angle (uncontrolled)
- `min`: `number` - Minimum angle (default: 0)
- `max`: `number` - Maximum angle (default: 360)
- `step`: `number` - Step increment (default: 1)
- `onValueChange`: `(value: number) => void` - Value change handler

### Accessibility
- Full keyboard navigation with Arrow keys
- Screen reader announcements for current value
- Focus management and ARIA labels
