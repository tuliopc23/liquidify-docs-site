# ScrollArea

A custom scrollable container component that provides styled scrollbars and precise viewport control. Perfect for content viewers, chat interfaces, data tables, or any scenario requiring custom scrolling behavior with consistent cross-browser styling.

## Installation

```bash
npm install @liquidify/components
```

## Import

```tsx
import { ScrollArea } from "@liquidify/components/scrollArea";
```

## Usage

### Chat Interface

```tsx
import { useState, useRef, useEffect } from "react";
import { ScrollArea } from "@liquidify/components/scrollArea";

export default function ChatInterface() {
  const [messages, setMessages] = useState([
    { id: 1, text: "Hello! How can I help you today?", sender: "bot", time: "10:00 AM" },
    { id: 2, text: "I need help with my account settings.", sender: "user", time: "10:01 AM" },
    { id: 3, text: "I'd be happy to help! Let me guide you through the account settings.", sender: "bot", time: "10:01 AM" },
    { id: 4, text: "First, click on your profile picture in the top right corner.", sender: "bot", time: "10:02 AM" },
  ]);
  const [newMessage, setNewMessage] = useState("");
  const viewportRef = useRef(null);

  const addMessage = (text, sender) => {
    const message = {
      id: messages.length + 1,
      text,
      sender,
      time: new Date().toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' })
    };
    setMessages(prev => [...prev, message]);
  };

  const handleSendMessage = () => {
    if (newMessage.trim()) {
      addMessage(newMessage, "user");
      setNewMessage("");

      // Simulate bot response
      setTimeout(() => {
        addMessage("Thanks for your message! I'll get back to you shortly.", "bot");
      }, 1000);
    }
  };

  const handleKeyPress = (e) => {
    if (e.key === "Enter") {
      handleSendMessage();
    }
  };

  // Auto-scroll to bottom when new messages arrive
  useEffect(() => {
    if (viewportRef.current) {
      viewportRef.current.scrollTop = viewportRef.current.scrollHeight;
    }
  }, [messages]);

  return (
    <div style={{ height: "500px", border: "1px solid #ccc", borderRadius: "8px", display: "flex", flexDirection: "column" }}>
      <div style={{ padding: "1rem", borderBottom: "1px solid #dee2e6", backgroundColor: "#f8f9fa" }}>
        <h3 style={{ margin: 0 }}>Customer Support Chat</h3>
      </div>

      <ScrollArea.Root style={{ flex: 1 }}>
        <ScrollArea.Viewport ref={viewportRef} style={{ height: "100%", padding: "1rem" }}>
          <div style={{ display: "flex", flexDirection: "column", gap: "1rem" }}>
            {messages.map(message => (
              <div key={message.id} style={{
                display: "flex",
                justifyContent: message.sender === "user" ? "flex-end" : "flex-start"
              }}>
                <div style={{
                  maxWidth: "70%",
                  padding: "0.75rem",
                  borderRadius: "8px",
                  backgroundColor: message.sender === "user" ? "#007bff" : "#e9ecef",
                  color: message.sender === "user" ? "white" : "#333",
                  position: "relative"
                }}>
                  <p style={{ margin: 0 }}>{message.text}</p>
                  <small style={{
                    fontSize: "0.7rem",
                    opacity: 0.7,
                    display: "block",
                    marginTop: "0.25rem"
                  }}>
                    {message.time}
                  </small>
                </div>
              </div>
            ))}
          </div>
        </ScrollArea.Viewport>

        <ScrollArea.Scrollbar orientation="vertical">
          <ScrollArea.Thumb />
        </ScrollArea.Scrollbar>
      </ScrollArea.Root>

      <div style={{ padding: "1rem", borderTop: "1px solid #dee2e6", display: "flex", gap: "0.5rem" }}>
        <input
          type="text"
          value={newMessage}
          onChange={(e) => setNewMessage(e.target.value)}
          onKeyPress={handleKeyPress}
          placeholder="Type your message..."
          style={{
            flex: 1,
            padding: "0.5rem",
            border: "1px solid #ccc",
            borderRadius: "4px"
          }}
        />
        <button
          onClick={handleSendMessage}
          style={{
            padding: "0.5rem 1rem",
            backgroundColor: "#007bff",
            color: "white",
            border: "none",
            borderRadius: "4px",
            cursor: "pointer"
          }}
        >
          Send
        </button>
      </div>
    </div>
  );
}
```

### Data Table with Scroll

```tsx

export default function DataTableScroll() {

      </ScrollArea.Root>
```

### Code Viewer

```tsx
import { ScrollArea } from "@liquidify/components/scrollArea";

export default function CodeViewer() {
  const code = `import React, { useState, useEffect } from 'react';

function DataFetcher({ url }) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        setLoading(true);
        const response = await fetch(url);
        if (!response.ok) {
          throw new Error('Network response was not ok');
        }
        const result = await response.json();
        setData(result);
      } catch (err) {
        setError(err.message);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, [url]);

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error}</div>;

  return (
    <div>
      <h2>Data:</h2>
      <pre>{JSON.stringify(data, null, 2)}</pre>
    </div>
  );
}

export default DataFetcher;`;

  return (
    <div style={{ height: "400px", border: "1px solid #ccc", borderRadius: "8px" }}>
      <div style={{
        padding: "1rem",
        backgroundColor: "#f8f9fa",

        display: "flex",
        justifyContent: "space-between",

        <h3 style={{ margin: 0 }}>DataFetcher.tsx</h3>

            margin: 0,

          }}>

          </pre>

        <ScrollArea.Scrollbar orientation="vertical">
          <ScrollArea.Thumb />
        </ScrollArea.Scrollbar>

        <ScrollArea.Scrollbar orientation="horizontal">
          <ScrollArea.Thumb />
        </ScrollArea.Scrollbar>

        <ScrollArea.Corner />
      </ScrollArea.Root>
```

## API Reference

### ScrollArea.Root

The root container for the scroll area component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The scroll area content |
| All standard div props | - | - | Standard HTML div attributes |

### ScrollArea.Viewport

The scrollable viewport that contains the content.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The viewport content |
| All standard div props | - | - | Standard HTML div attributes |

### ScrollArea.Scrollbar

The scrollbar component for controlling scroll position.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | The scrollbar orientation |
| `children` | `ReactNode` | - | The scrollbar content (usually Thumb) |

### ScrollArea.Thumb

The draggable thumb component of the scrollbar.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The thumb content |

### ScrollArea.Corner

The corner component where horizontal and vertical scrollbars meet.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The corner content |

## Accessibility

The ScrollArea component includes comprehensive accessibility features:

- **ARIA Roles**: Implements proper scrollbar and scrollable region roles
- **Keyboard Navigation**: Full keyboard support for scrolling
- **Screen Reader Support**: Announces scroll position and content
- **Focus Management**: Proper focus handling within scrollable areas
- **High Contrast**: Compatible with high contrast themes

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Arrow Up/Down` | Scroll vertically |
| `Arrow Left/Right` | Scroll horizontally |
| `Page Up/Down` | Scroll by page |
| `Home` | Scroll to top |
| `End` | Scroll to bottom |
| `Tab` | Navigate through focusable elements |

## Best Practices

1. **Consistent Styling**: Use consistent scrollbar styling across your application
2. **Performance**: Avoid putting large amounts of content in scroll areas
3. **Touch Support**: Ensure scroll areas work well on touch devices
4. **Visual Feedback**: Provide clear visual indicators for scrollable content
5. **Keyboard Navigation**: Ensure all scrollable content is keyboard accessible
6. **Loading States**: Handle loading states gracefully for dynamic content
7. **Responsive Design**: Consider how scroll areas behave on different screen sizes
8. **Content Organization**: Use scroll areas to organize long content sections
