# Checkbox

Binary input control for selecting multiple options from a list. Essential for forms, settings, and multi-selection interfaces.

## Import

```tsx
import { Checkbox } from "liquidify-react/checkbox"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<Checkbox.Root>
			<Checkbox.Control>
				<Checkbox.Indicator>
					<CheckIcon />
				</Checkbox.Indicator>
			</Checkbox.Control>
			<Checkbox.Label>Accept terms and conditions</Checkbox.Label>
		</Checkbox.Root>
	)
}
```

## Controlled State

```tsx
import { useState } from "react"

export default function ControlledExample() {
	const [checked, setChecked] = useState(false)
	
	return (
		<Checkbox.Root checked={checked} onCheckedChange={setChecked}>
			<Checkbox.Control>
				<Checkbox.Indicator>
					<CheckIcon />
				</Checkbox.Indicator>
			</Checkbox.Control>
			<Checkbox.Label>
				Newsletter subscription {checked ? "(subscribed)" : "(not subscribed)"}
			</Checkbox.Label>
		</Checkbox.Root>
	)
}
```

## Indeterminate State

```tsx
import { useState } from "react"

export default function IndeterminateExample() {
	const [items, setItems] = useState([
		{ id: "1", checked: true, label: "Item 1" },
		{ id: "2", checked: false, label: "Item 2" },
		{ id: "3", checked: true, label: "Item 3" }
	])
	
	const allChecked = items.every(item => item.checked)
	const someChecked = items.some(item => item.checked)
	
	return (
		<div>
			<Checkbox.Root 
				checked={allChecked}
				indeterminate={someChecked && !allChecked}
				onCheckedChange={(checked) => {
					setItems(items.map(item => ({ ...item, checked })))
				}}
			>
				<Checkbox.Control>
					<Checkbox.Indicator>
						{someChecked && !allChecked ? <MinusIcon /> : <CheckIcon />}
					</Checkbox.Indicator>
				</Checkbox.Control>
				<Checkbox.Label>Select All</Checkbox.Label>
			</Checkbox.Root>
			
			<div style={{ marginLeft: "1.5rem", marginTop: "0.5rem" }}>
				{items.map(item => (
					<Checkbox.Root 
						key={item.id}
						checked={item.checked}
						onCheckedChange={(checked) => {
							setItems(items.map(i => 
								i.id === item.id ? { ...i, checked } : i
							))
						}}
					>
						<Checkbox.Control>
							<Checkbox.Indicator>
								<CheckIcon />
							</Checkbox.Indicator>
						</Checkbox.Control>
						<Checkbox.Label>{item.label}</Checkbox.Label>
					</Checkbox.Root>
				))}
			</div>
		</div>
	)
}
```

## API Reference

### Root Props
- `checked`: `boolean` - Controlled checked state
- `defaultChecked`: `boolean` - Default checked state (uncontrolled)
- `indeterminate`: `boolean` - Indeterminate state
- `disabled`: `boolean` - Disable interaction
- `required`: `boolean` - Required field validation
- `onCheckedChange`: `(checked: boolean) => void` - State change handler

### Accessibility
- Full keyboard navigation with Space key
- Proper ARIA attributes and roles
- Screen reader announcements for state changes
- Label association for click targeting
