# Button

Interactive elements that trigger actions when clicked. Essential for forms, navigation, and user interactions with multiple variants for different contexts.

## Import

```tsx
import { Button } from "liquidify-react/button"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Button variant="primary">Primary</Button>
			<Button variant="secondary">Secondary</Button>
			<Button variant="ghost">Ghost</Button>
		</div>
	)
}
```

## Sizes

```tsx
export default function SizesExample() {
	return (
		<div style={{ display: "flex", gap: "1rem", alignItems: "center" }}>
			<Button size="sm">Small</Button>
			<Button size="md">Medium</Button>
			<Button size="lg">Large</Button>
		</div>
	)
}
```

## With Icons

```tsx
export default function IconsExample() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Button>
				<PlusIcon />
				Add Item
			</Button>
			
			<Button variant="secondary">
				Download
				<DownloadIcon />
			</Button>
			
			<Button variant="ghost" size="sm">
				<SearchIcon />
			</Button>
		</div>
	)
}
```

## States

```tsx
export default function StatesExample() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Button>Default</Button>
			<Button disabled>Disabled</Button>
			<Button loading>Loading</Button>
		</div>
	)
}
```

## As Link

```tsx
export default function LinkExample() {
	return (
		<Button asChild>
			<a href="/dashboard">Go to Dashboard</a>
		</Button>
	)
}
```

## API Reference

### Props
- `variant`: `"primary" | "secondary" | "ghost" | "destructive"` - Visual style
- `size`: `"sm" | "md" | "lg"` - Button size
- `disabled`: `boolean` - Disable interaction
- `loading`: `boolean` - Show loading state
- `asChild`: `boolean` - Render as child element (for links)

### Accessibility
- Full keyboard navigation with Enter and Space
- Disabled state properly announced
- Loading state with aria-busy
- Focus management and visual indicators
