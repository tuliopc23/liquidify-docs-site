# RadioGroup

A set of radio buttons that allows users to select one option from multiple choices.

## Installation

```bash
bun add @liquidify/components
```

## Import

```tsx
import { RadioGroup } from "@liquidify/components/radio-group"
```

## Basic Usage

Create a simple radio group with multiple options:

```tsx
import { RadioGroup } from "@liquidify/components/radio-group"

export function BasicExample() {
  return (
    <RadioGroup.Root defaultValue="medium">
      <RadioGroup.Label>Size</RadioGroup.Label>
      <div style={{ display: "flex", flexDirection: "column", gap: "8px" }}>
        <RadioGroup.Item value="small">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>Small</RadioGroup.ItemText>
        </RadioGroup.Item>
        <RadioGroup.Item value="medium">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>Medium</RadioGroup.ItemText>
        </RadioGroup.Item>
        <RadioGroup.Item value="large">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>Large</RadioGroup.ItemText>
        </RadioGroup.Item>
      </div>
    </RadioGroup.Root>
  )
}
```

## Controlled Radio Group

Control the selected value with state:

```tsx
import { useState } from "react"
import { RadioGroup } from "@liquidify/components/radio-group"

export function ControlledExample() {
  const [value, setValue] = useState("credit-card")

  return (
    <RadioGroup.Root value={value} onValueChange={(details) => setValue(details.value)}>
      <RadioGroup.Label>Payment Method</RadioGroup.Label>
      <div style={{ display: "flex", flexDirection: "column", gap: "12px" }}>
        <RadioGroup.Item value="credit-card">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>Credit Card</RadioGroup.ItemText>
        </RadioGroup.Item>
        <RadioGroup.Item value="paypal">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>PayPal</RadioGroup.ItemText>
        </RadioGroup.Item>
        <RadioGroup.Item value="bank-transfer">
          <RadioGroup.ItemControl />
          <RadioGroup.ItemText>Bank Transfer</RadioGroup.ItemText>
        </RadioGroup.Item>
      </div>
      <p>Selected: {value}</p>
    </RadioGroup.Root>
  )
}
```

## Validation and Error States

Show validation errors for radio groups:

```tsx
import { useState } from "react"
import { RadioGroup } from "@liquidify/components/radio-group"
import { Button } from "@liquidify/components/button"

export function ValidationExample() {
  const [value, setValue] = useState("")
  const [error, setError] = useState("")

  const handleSubmit = () => {
    if (!value) {
      setError("Please select an option")
    } else {
      setError("")
      console.log("Selected:", value)
    }
  }

  return (
    <div>
      <RadioGroup.Root 
        value={value} 
        onValueChange={(details) => {
          setValue(details.value)
          setError("")
        }}
        invalid={!!error}
      >
        <RadioGroup.Label>Required: Choose your preference</RadioGroup.Label>
        <div style={{ display: "flex", flexDirection: "column", gap: "8px" }}>
          <RadioGroup.Item value="option1">
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>Option 1</RadioGroup.ItemText>
          </RadioGroup.Item>
          <RadioGroup.Item value="option2">
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>Option 2</RadioGroup.ItemText>
          </RadioGroup.Item>
          <RadioGroup.Item value="option3">
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>Option 3</RadioGroup.ItemText>
          </RadioGroup.Item>
        </div>
        {error && (
          <p style={{ color: "red", fontSize: "14px", margin: "8px 0 0 0" }}>
            {error}
          </p>
        )}
      </RadioGroup.Root>
      
      <Button onClick={handleSubmit} style={{ marginTop: "16px" }}>
        Submit
      </Button>
    </div>
  )
}
```

## API Reference

### RadioGroup.Root

The root container for the radio group.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled selected value |
| `defaultValue` | `string` | - | Default selected value |
| `onValueChange` | `(details) => void` | - | Callback when selection changes |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Layout orientation |
| `disabled` | `boolean` | `false` | Whether the group is disabled |
| `invalid` | `boolean` | `false` | Whether the group is invalid |
| `required` | `boolean` | `false` | Whether selection is required |

### RadioGroup.Label

The label for the radio group.

### RadioGroup.Item

Individual radio button item.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Value of the radio item (required) |
| `disabled` | `boolean` | `false` | Whether the item is disabled |

### RadioGroup.ItemControl

The visual radio button control.

### RadioGroup.ItemText

The text label for the radio item.

## Accessibility

The RadioGroup component follows WAI-ARIA guidelines:

- Uses `role="radiogroup"` for the container
- Implements `role="radio"` for individual items
- Supports keyboard navigation with arrow keys
- Manages focus within the group
- Provides proper `aria-checked` states
- Associates labels with `aria-labelledby`
- Supports `aria-describedby` for error messages

## Best Practices

- Always provide clear, descriptive labels for each option
- Use radio groups when only one selection is allowed
- Consider using checkboxes for multiple selections instead
- Provide a default selection when appropriate
- Group related options logically
- Use validation messages to guide users
- Ensure sufficient contrast for disabled states
- Test keyboard navigation and screen reader compatibility
