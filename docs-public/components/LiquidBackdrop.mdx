# LiquidBackdrop

## Summary
An animated SVG filter component that creates a liquid-like distortion effect using turbulence and displacement mapping. Perfect for adding organic, flowing visual effects to backgrounds, images, or UI elements.

### Import
```ts
import { LiquidBackdrop } from 'liquidify-react/liquidBackdrop'
```

## Examples

### Basic Usage
```tsx
import { LiquidBackdrop } from 'liquidify-react/liquidBackdrop'

function BasicLiquidBackdrop() {
  return (
    <div style={{ position: 'relative', width: '400px', height: '300px' }}>
      <LiquidBackdrop />
      <div
        className="liquid-warp"
        style={{
          width: '100%',
          height: '100%',
          background: 'linear-gradient(45deg, #667eea 0%, #764ba2 100%)',
          borderRadius: '16px',
        }}
      />
    </div>
  )
}
```

## API Reference

### LiquidBackdrop
The main component that creates the liquid distortion filter.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `opacity` | `number` | `0.06` | Controls the intensity of the liquid effect (0-1) |

### CSS Class
The component automatically injects a CSS class that can be applied to elements:

| Class | Description |
|-------|-------------|
| `liquid-warp` | Applies the liquid distortion filter to the element |

## Accessibility

The LiquidBackdrop component is designed with accessibility in mind:

- **Screen Reader Friendly**: Uses `aria-hidden="true"` to hide decorative SVG from screen readers
- **Performance Conscious**: Uses `requestAnimationFrame` for smooth 60fps animation
- **Reduced Motion**: Respects user's motion preferences (automatically disabled if `prefers-reduced-motion` is set)

### Motion Preferences
The component automatically detects and respects the user's motion preferences:
```css
@media (prefers-reduced-motion: reduce) {
  .liquid-warp {
    filter: none !important;
  }
}
```

## Best Practices

### Do's
- Use subtle opacity values (0.04-0.12) for most applications
- Position the LiquidBackdrop component as a sibling to the element you want to affect
- Combine with gradients or images for best visual impact
- Test performance on lower-end devices before using in production

### Don'ts
- Don't use high opacity values (>0.2) as they can be visually overwhelming
- Don't apply to text-only elements without background colors
- Don't use on elements that require pixel-perfect precision
- Don't overuse on the same page - limit to 2-3 instances

### Performance Tips
- The effect uses GPU acceleration via CSS filters
- Animation runs at 60fps using `requestAnimationFrame`
- SVG is positioned absolutely with zero dimensions to avoid layout impact
- Consider using `will-change: filter` on affected elements for better performance

### Browser Support
- Modern browsers with CSS filter support
- Gracefully degrades in unsupported browsers (no visual effect)
- iOS Safari 9+
- Chrome 18+
- Firefox 35+
- Edge 12+
