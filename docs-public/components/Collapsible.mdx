# Collapsible

Show and hide content sections with smooth animations. Perfect for FAQ sections, navigation menus, and progressive disclosure.

## Import

```tsx
import { Collapsible } from "liquidify-react/collapsible"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<Collapsible.Root>
			<Collapsible.Trigger>
				Show more details
				<ChevronDownIcon />
			</Collapsible.Trigger>
			<Collapsible.Content>
				<p>This content can be shown or hidden with smooth animation.</p>
				<p>Perfect for revealing additional information on demand.</p>
			</Collapsible.Content>
		</Collapsible.Root>
	)
}
```

## Controlled State

```tsx
import { useState } from "react"

export default function ControlledExample() {
	const [open, setOpen] = useState(false)
	
	return (
		<div>
			<p>Status: {open ? "Open" : "Closed"}</p>
			<Collapsible.Root open={open} onOpenChange={setOpen}>
				<Collapsible.Trigger>
					{open ? "Hide" : "Show"} details
				</Collapsible.Trigger>
				<Collapsible.Content>
					<div style={{ padding: "1rem", background: "#f5f5f5" }}>
						<h4>Collapsible Content</h4>
						<p>This content visibility is controlled by external state.</p>
					</div>
				</Collapsible.Content>
			</Collapsible.Root>
		</div>
	)
}
```

## Multiple Collapsibles

```tsx
export default function MultipleExample() {
	const sections = [
		{
			id: "section1",
			title: "Getting Started",
			content: "Learn the basics of using our component library."
		},
		{
			id: "section2", 
			title: "Advanced Usage",
			content: "Explore advanced patterns and customization options."
		},
		{
			id: "section3",
			title: "Best Practices", 
			content: "Follow these guidelines for optimal performance."
		}
	]
	
	return (
		<div style={{ display: "flex", flexDirection: "column", gap: "1rem" }}>
			{sections.map(section => (
				<Collapsible.Root key={section.id}>
					<Collapsible.Trigger 
						style={{ 
							width: "100%", 
							textAlign: "left",
							padding: "1rem",
							border: "1px solid #ccc",
							borderRadius: "8px"
						}}
					>
						{section.title}
					</Collapsible.Trigger>
					<Collapsible.Content>
						<div style={{ padding: "1rem", border: "1px solid #eee" }}>
							{section.content}
						</div>
					</Collapsible.Content>
				</Collapsible.Root>
			))}
		</div>
	)
}
```

## With Custom Animation

```tsx
export default function AnimationExample() {
	return (
		<Collapsible.Root>
			<Collapsible.Trigger>
				Smooth Animation
			</Collapsible.Trigger>
			<Collapsible.Content
				style={{
					transition: "all 0.3s ease-in-out",
					overflow: "hidden"
				}}
			>
				<div style={{ padding: "1rem" }}>
					<p>Content animates smoothly when expanding and collapsing.</p>
					<p>The animation respects user motion preferences.</p>
				</div>
			</Collapsible.Content>
		</Collapsible.Root>
	)
}
```

## API Reference

### Root Props
- `open`: `boolean` - Controlled open state
- `defaultOpen`: `boolean` - Default open state (uncontrolled)
- `onOpenChange`: `(open: boolean) => void` - State change handler
- `disabled`: `boolean` - Disable interaction

### Accessibility
- ARIA expanded states for screen readers
- Keyboard navigation with Enter and Space
- Proper focus management
- Respects prefers-reduced-motion settings
