# Fieldset

Groups related form controls with a legend for semantic organization and accessibility. Essential for complex forms and settings panels.

## Import

```tsx
import { Fieldset } from "liquidify-react/fieldset"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<Fieldset.Root>
			<Fieldset.Legend>Personal Information</Fieldset.Legend>
			<Fieldset.HelperText>Please provide your basic details</Fieldset.HelperText>
			
			<div style={{ display: "grid", gap: "1rem" }}>
				<Field.Root>
					<Field.Label>First Name</Field.Label>
					<Field.Input placeholder="Enter first name" />
				</Field.Root>
				
				<Field.Root>
					<Field.Label>Last Name</Field.Label>
					<Field.Input placeholder="Enter last name" />
				</Field.Root>
				
				<Field.Root>
					<Field.Label>Email</Field.Label>
					<Field.Input type="email" placeholder="Enter email" />
				</Field.Root>
			</div>
		</Fieldset.Root>
	)
}
```

## Disabled Fieldset

```tsx
export default function DisabledExample() {
	return (
		<Fieldset.Root disabled>
			<Fieldset.Legend>Account Settings</Fieldset.Legend>
			<Fieldset.HelperText>These settings cannot be modified</Fieldset.HelperText>
			
			<div style={{ display: "grid", gap: "1rem" }}>
				<Field.Root>
					<Field.Label>Username</Field.Label>
					<Field.Input value="john_doe" readOnly />
				</Field.Root>
				
				<Field.Root>
					<Field.Label>Account Type</Field.Label>
					<Field.Select disabled>
						<option>Premium</option>
					</Field.Select>
				</Field.Root>
			</div>
		</Fieldset.Root>
	)
}
```

## Validation Error

```tsx
export default function ValidationExample() {
	return (
		<Fieldset.Root invalid>
			<Fieldset.Legend>Billing Address</Fieldset.Legend>
			<Fieldset.ErrorText>Please correct the errors below</Fieldset.ErrorText>
			
			<div style={{ display: "grid", gap: "1rem" }}>
				<Field.Root invalid>
					<Field.Label>Street Address</Field.Label>
					<Field.Input value="" />
					<Field.ErrorText>Street address is required</Field.ErrorText>
				</Field.Root>
				
				<div style={{ display: "grid", gridTemplateColumns: "2fr 1fr", gap: "1rem" }}>
					<Field.Root>
						<Field.Label>City</Field.Label>
						<Field.Input placeholder="Enter city" />
					</Field.Root>
					
					<Field.Root invalid>
						<Field.Label>Postal Code</Field.Label>
						<Field.Input value="123" />
						<Field.ErrorText>Invalid format</Field.ErrorText>
					</Field.Root>
				</div>
			</div>
		</Fieldset.Root>
	)
}
```

## Multiple Fieldsets

```tsx
export default function MultipleExample() {
	return (
		<form style={{ display: "grid", gap: "2rem" }}>
			<Fieldset.Root>
				<Fieldset.Legend>Contact Information</Fieldset.Legend>
				<div style={{ display: "grid", gap: "1rem" }}>
					<Field.Root>
						<Field.Label>Email</Field.Label>
						<Field.Input type="email" />
					</Field.Root>
					<Field.Root>
						<Field.Label>Phone</Field.Label>
						<Field.Input type="tel" />
					</Field.Root>
				</div>
			</Fieldset.Root>
			
			<Fieldset.Root>
				<Fieldset.Legend>Preferences</Fieldset.Legend>
				<Fieldset.HelperText>Customize your experience</Fieldset.HelperText>
				<div style={{ display: "grid", gap: "1rem" }}>
					<label style={{ display: "flex", alignItems: "center", gap: "0.5rem" }}>
						<input type="checkbox" />
						Email notifications
					</label>
					<label style={{ display: "flex", alignItems: "center", gap: "0.5rem" }}>
						<input type="checkbox" />
						SMS notifications
					</label>
				</div>
			</Fieldset.Root>
			
			<Fieldset.Root>
				<Fieldset.Legend>Security</Fieldset.Legend>
				<div style={{ display: "grid", gap: "1rem" }}>
					<Field.Root>
						<Field.Label>Current Password</Field.Label>
						<Field.Input type="password" />
					</Field.Root>
					<Field.Root>
						<Field.Label>New Password</Field.Label>
						<Field.Input type="password" />
					</Field.Root>
				</div>
			</Fieldset.Root>
		</form>
	)
}
```

## API Reference

### Root Props
- `disabled`: `boolean` - Disable all contained form controls
- `invalid`: `boolean` - Mark fieldset as having validation errors
- `form`: `string` - Associate with form element by ID

### Accessibility
- Semantic grouping with `<fieldset>` element
- Proper legend association for screen readers
- Automatic disability inheritance for child controls
- Keyboard navigation support
