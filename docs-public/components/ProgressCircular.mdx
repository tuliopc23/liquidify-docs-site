# ProgressCircular

A circular progress indicator component that displays completion status with smooth animations. Perfect for loading states, file uploads, task completion tracking, or any scenario requiring visual progress feedback in a compact circular format.

## Installation

```bash
npm install @liquidify/components
```

## Import

```tsx
import { ProgressCircular } from "@liquidify/components/progressCircular";
```

## Usage

### Basic Progress Circular

```tsx
import { useState } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function BasicProgressCircular() {
  const [progress, setProgress] = useState(0);

  const incrementProgress = () => {
    setProgress(prev => Math.min(100, prev + 10));
  };

  const resetProgress = () => {
    setProgress(0);
  };

  return (
    <div style={{ textAlign: "center", padding: "2rem" }}>
      <ProgressCircular value={progress} label={`${progress}%`} />

      <div style={{ marginTop: "2rem" }}>
        <button onClick={incrementProgress} style={{ marginRight: "1rem" }}>
          Increase Progress
        </button>
        <button onClick={resetProgress}>
          Reset
        </button>
      </div>

      <p style={{ marginTop: "1rem", color: "#666" }}>
        Current progress: {progress}%
      </p>
    </div>
  );
}
```

### File Upload Progress

```tsx
import { useState, useEffect } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function FileUploadProgress() {
  const [uploadProgress, setUploadProgress] = useState(0);
  const [isUploading, setIsUploading] = useState(false);
  const [uploadedFiles, setUploadedFiles] = useState([]);

  const simulateUpload = (fileName) => {
    setIsUploading(true);
    setUploadProgress(0);

    const interval = setInterval(() => {
      setUploadProgress(prev => {
        if (prev >= 100) {
          clearInterval(interval);
          setIsUploading(false);
          setUploadedFiles(prevFiles => [...prevFiles, fileName]);
          return 100;
        }
        return prev + Math.random() * 15;
      });
    }, 200);
  };

  const handleFileUpload = (event) => {
    const file = event.target.files[0];
    if (file) {
      simulateUpload(file.name);
    }
  };

  return (
    <div style={{ maxWidth: "500px", margin: "0 auto", padding: "2rem" }}>
      <h3>File Upload</h3>

      <div style={{ marginBottom: "2rem" }}>
        <input
          type="file"
          onChange={handleFileUpload}
          disabled={isUploading}
          style={{ marginBottom: "1rem" }}
        />

        {isUploading && (
          <div style={{ textAlign: "center", padding: "2rem" }}>
            <ProgressCircular
              value={uploadProgress}
              size="lg"
              label={`Uploading... ${Math.round(uploadProgress)}%`}
            />
            <p style={{ marginTop: "1rem", color: "#666" }}>
              Please wait while your file is being uploaded
            </p>
          </div>
        )}
      </div>

      {uploadedFiles.length > 0 && (
        <div>
          <h4>Uploaded Files</h4>
          <ul>
            {uploadedFiles.map((file, index) => (
              <li key={index} style={{ color: "#28a745" }}>
                ✅ {file}
              </li>
            ))}
          </ul>
        </div>
      )}
    </div>
  );
}
```

### Loading States

```tsx
import { useState, useEffect } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function LoadingStates() {
  const [loadingStates, setLoadingStates] = useState({
    api: 0,
    database: 0,
    cache: 0
  });

  const [completed, setCompleted] = useState([]);

  useEffect(() => {
    const services = ['api', 'database', 'cache'];

    services.forEach((service, index) => {
      setTimeout(() => {
        const interval = setInterval(() => {
          setLoadingStates(prev => {
            const newValue = prev[service] + Math.random() * 20;
            if (newValue >= 100) {
              clearInterval(interval);
              setCompleted(prevCompleted => [...prevCompleted, service]);
              return { ...prev, [service]: 100 };
            }
            return { ...prev, [service]: newValue };
          });
        }, 300);
      }, index * 1000); // Stagger the start times
    });
  }, []);

  const allCompleted = completed.length === 3;

  return (
    <div style={{ maxWidth: "600px", margin: "0 auto", padding: "2rem" }}>
      <h3>System Initialization</h3>

      <div style={{ display: "grid", gridTemplateColumns: "repeat(auto-fit, minmax(150px, 1fr))", gap: "2rem", marginBottom: "2rem" }}>

          </p>

          <ProgressCircular
            value={loadingStates.database}

          </p>
        </div>
      )}
    </div>
  );
```

### Task Completion Tracker

```tsx
import { useState } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function TaskCompletionTracker() {
  const [tasks, setTasks] = useState([
    { id: 1, title: "Design mockups", completed: true },
    { id: 2, title: "Write documentation", completed: false },
    { id: 3, title: "Implement features", completed: false },
    { id: 4, title: "Run tests", completed: false },
    { id: 5, title: "Deploy to production", completed: false }
  ]);

  const completedTasks = tasks.filter(task => task.completed).length;
  const totalTasks = tasks.length;
  const progress = (completedTasks / totalTasks) * 100;

  const toggleTask = (taskId) => {
    setTasks(prevTasks =>
      prevTasks.map(task =>
        task.id === taskId ? { ...task, completed: !task.completed } : task
      )
    );
  };

  return (
    <div style={{ maxWidth: "500px", margin: "0 auto", padding: "2rem" }}>
      <h3>Project Progress</h3>

      <div style={{ textAlign: "center", marginBottom: "2rem" }}>
        <ProgressCircular
          value={progress}
          size="lg"
          label={`${completedTasks}/${totalTasks} Tasks`}
        />
        <p style={{ marginTop: "1rem", color: "#666" }}>
          {Math.round(progress)}% Complete
        </p>
      </div>

      <div>
        <h4>Task List</h4>
        <div style={{ display: "flex", flexDirection: "column", gap: "0.5rem" }}>
          {tasks.map(task => (
            <div
              key={task.id}
              style={{
                display: "flex",
                alignItems: "center",
                padding: "0.75rem",
                border: "1px solid #dee2e6",
                borderRadius: "8px",
                backgroundColor: task.completed ? "#d4edda" : "#fff",
                cursor: "pointer"
              }}
              onClick={() => toggleTask(task.id)}
            >
              <input
                type="checkbox"
                checked={task.completed}
                onChange={() => toggleTask(task.id)}
                style={{ marginRight: "0.75rem" }}
              />
              <span style={{
                textDecoration: task.completed ? "line-through" : "none",
                color: task.completed ? "#155724" : "#333"
              }}>
                {task.title}
              </span>
              {task.completed && (
                <span style={{ marginLeft: "auto", color: "#28a745" }}>✓</span>
              )}
            </div>
          ))}
        </div>
      </div>
    </div>
  );
}
```

### Countdown Timer

```tsx
import { useState, useEffect } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function CountdownTimer() {
  const [timeLeft, setTimeLeft] = useState(60); // 60 seconds
  const [isRunning, setIsRunning] = useState(false);
  const totalTime = 60;

  useEffect(() => {
    let interval;
    if (isRunning && timeLeft > 0) {
      interval = setInterval(() => {
        setTimeLeft(prev => {
          if (prev <= 1) {
            setIsRunning(false);
            return 0;
          }
          return prev - 1;
        });
      }, 1000);
    }

    return () => clearInterval(interval);
  }, [isRunning, timeLeft]);

  const progress = ((totalTime - timeLeft) / totalTime) * 100;
  const minutes = Math.floor(timeLeft / 60);
  const seconds = timeLeft % 60;

  const startTimer = () => setIsRunning(true);
  const pauseTimer = () => setIsRunning(false);
  const resetTimer = () => {
    setIsRunning(false);
    setTimeLeft(totalTime);
  };

  return (
    <div style={{ textAlign: "center", padding: "2rem" }}>
      <h3>Pomodoro Timer</h3>

      <div style={{ marginBottom: "2rem" }}>
        <ProgressCircular
          value={progress}
          size="lg"
          label={`${minutes}:${seconds.toString().padStart(2, '0')}`}
        />
      </div>

      <div style={{ marginBottom: "2rem" }}>
        <div style={{ fontSize: "2rem", fontWeight: "bold", marginBottom: "0.5rem" }}>
          {minutes}:{seconds.toString().padStart(2, '0')}
        </div>
        <div style={{ color: "#666" }}>
          {timeLeft === 0 ? "Time's up!" : isRunning ? "Focus time" : "Paused"}
        </div>
      </div>

      <div>
        {!isRunning && timeLeft > 0 && (
          <button
            onClick={startTimer}
            style={{
              padding: "0.75rem 1.5rem",
              backgroundColor: "#28a745",
              color: "white",
              border: "none",
              borderRadius: "4px",
              cursor: "pointer",
              marginRight: "0.5rem"
            }}
          >
            Start
          </button>
        )}

        {isRunning && (
          <button
            onClick={pauseTimer}
            style={{
              padding: "0.75rem 1.5rem",
              backgroundColor: "#ffc107",
              color: "black",
              border: "none",
              borderRadius: "4px",
              cursor: "pointer",
              marginRight: "0.5rem"
            }}
          >
            Pause
          </button>
        )}

        <button
          onClick={resetTimer}
          style={{
            padding: "0.75rem 1.5rem",
            backgroundColor: "#6c757d",
            color: "white",
            border: "none",
            borderRadius: "4px",
            cursor: "pointer"
          }}
        >
          Reset
        </button>
      </div>
    </div>
  );
}
```

### Multiple Progress Indicators

```tsx
import { useState, useEffect } from "react";
import { ProgressCircular } from "@liquidify/components/progressCircular";

export default function MultipleProgressIndicators() {
  const [progresses, setProgresses] = useState({
    download: 0,
    upload: 0,
    processing: 0,
    validation: 0
  });

  const [activeProcess, setActiveProcess] = useState("download");

  useEffect(() => {
    const processes = ["download", "upload", "processing", "validation"];

    processes.forEach((process, index) => {
      setTimeout(() => {
        const interval = setInterval(() => {
          setProgresses(prev => {
            const newValue = prev[process] + Math.random() * 8;
            if (newValue >= 100) {
              clearInterval(interval);
              if (index < processes.length - 1) {
                setActiveProcess(processes[index + 1]);
              }
              return { ...prev, [process]: 100 };
            }
            return { ...prev, [process]: newValue };
          });
        }, 150);
      }, index * 2000); // Stagger the processes
    });
  }, []);

  const processLabels = {
    download: "Downloading files",
    upload: "Uploading data",
    processing: "Processing information",
    validation: "Validating results"
  };

  const isProcessActive = (process) => {
    const processes = ["download", "upload", "processing", "validation"];
    const currentIndex = processes.indexOf(activeProcess);
    const processIndex = processes.indexOf(process);
    return processIndex <= currentIndex;
  };

  return (
    <div style={{ maxWidth: "700px", margin: "0 auto", padding: "2rem" }}>
      <h3>Data Pipeline Progress</h3>

      <div style={{ display: "grid", gridTemplateColumns: "repeat(auto-fit, minmax(150px, 1fr))", gap: "2rem" }}>
        {Object.entries(progresses).map(([key, value]) => (
          <div key={key} style={{ textAlign: "center" }}>
            <ProgressCircular
              value={value}
              size="md"
              label={processLabels[key]}
            />
            <div style={{
              marginTop: "0.5rem",
              fontSize: "0.9rem",
              color: isProcessActive(key) ? "#007bff" : "#666"
            }}>
              {Math.round(value)}%
            </div>
            <div style={{
              fontSize: "0.8rem",
              color: value === 100 ? "#28a745" : isProcessActive(key) ? "#007bff" : "#ccc"
            }}>
              {value === 100 ? "✓ Complete" : isProcessActive(key) ? "In Progress" : "Pending"}
            </div>
          </div>
        ))}
      </div>

      <div style={{ marginTop: "2rem", textAlign: "center" }}>
        <div style={{ fontSize: "1.2rem", fontWeight: "bold", marginBottom: "0.5rem" }}>
          Overall Progress: {Math.round(Object.values(progresses).reduce((a, b) => a + b, 0) / 4)}%
        </div>
        <ProgressCircular
          value={Object.values(progresses).reduce((a, b) => a + b, 0) / 4}
          size="lg"
          label="Pipeline Complete"
        />
      </div>
    </div>
  );
}
```

## API Reference

### ProgressCircular

The circular progress indicator component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | The current progress value |
| `max` | `number` | `100` | The maximum progress value |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | The size of the progress indicator |
| `label` | `string` | - | The label text to display |
| `children` | `ReactNode` | - | Custom content to display instead of label |
| All standard div props | - | - | Standard HTML div attributes |

## Accessibility

The ProgressCircular component includes comprehensive accessibility features:

- **ARIA Roles**: Implements proper progressbar role
- **ARIA Labels**: Automatic labeling for screen readers
- **Screen Reader Support**: Announces progress values and status
- **Keyboard Navigation**: Full keyboard support
- **High Contrast**: Compatible with high contrast themes

## Best Practices

1. **Clear Context**: Always provide context about what's being tracked
2. **Appropriate Size**: Choose size based on available space and importance
3. **Smooth Updates**: Update progress values smoothly to avoid jarring changes
4. **Completion Feedback**: Provide clear feedback when progress reaches 100%
5. **Loading States**: Use indeterminate progress for unknown durations
6. **Performance**: Avoid updating progress too frequently to prevent performance issues
7. **User Feedback**: Combine with text labels for better accessibility
8. **Consistent Styling**: Maintain consistent progress indicator styling across your app
