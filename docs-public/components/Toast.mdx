# Toast

A temporary notification component that displays brief, auto-dismissing messages to inform users about the outcome of actions, system status, or important information without interrupting their workflow.

## Installation

```bash
npm install @liquidify/components
```

## Import

```tsx
import { Toast } from "@liquidify/components/toast";
```

## Usage

### Basic Toast

```tsx
import { useState } from "react";
import { Toast } from "@liquidify/components/toast";

export default function BasicToast() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>
        Show Toast
      </button>
      
      {isOpen && (
        <Toast.Root open={isOpen} onOpenChange={setIsOpen}>
          <Toast.Title>Success!</Toast.Title>
          <Toast.Description>
            Your changes have been saved successfully.
          </Toast.Description>
          <Toast.CloseTrigger>×</Toast.CloseTrigger>
        </Toast.Root>
      )}
    </div>
  );
}
```

### Toast with Different Status

```tsx
import { useState } from "react";
import { Toast } from "@liquidify/components/toast";

export default function StatusToasts() {
  const [activeToast, setActiveToast] = useState(null);

  const showToast = (status, title, description) => {
    setActiveToast({ status, title, description });
  };

  return (
    <div style={{ display: "flex", gap: "1rem" }}>
      <button onClick={() => showToast("success", "Success!", "Operation completed successfully.")}>
        Success Toast
      </button>
      <button onClick={() => showToast("error", "Error!", "Something went wrong. Please try again.")}>
        Error Toast
      </button>
      <button onClick={() => showToast("warning", "Warning!", "Please review your input before proceeding.")}>
        Warning Toast
      </button>
      <button onClick={() => showToast("info", "Info", "Here's some helpful information.")}>
        Info Toast
      </button>
      
      {activeToast && (
        <Toast.Root 
          open={!!activeToast} 
          onOpenChange={() => setActiveToast(null)}
          status={activeToast.status}
        >
          <Toast.Title>{activeToast.title}</Toast.Title>
          <Toast.Description>{activeToast.description}</Toast.Description>
          <Toast.CloseTrigger>×</Toast.CloseTrigger>
        </Toast.Root>
      )}
    </div>
  );
}
```

### Auto-dismissing Toast

```tsx
import { useState, useEffect } from "react";
import { Toast } from "@liquidify/components/toast";

export default function AutoDismissToast() {
  const [isOpen, setIsOpen] = useState(false);

  useEffect(() => {
    if (isOpen) {
      const timer = setTimeout(() => {
        setIsOpen(false);
      }, 3000); // Auto-dismiss after 3 seconds

      return () => clearTimeout(timer);
    }
  }, [isOpen]);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>
        Show Auto-dismiss Toast
      </button>
      
      {isOpen && (
        <Toast.Root open={isOpen} onOpenChange={setIsOpen}>
          <Toast.Title>Auto-dismiss</Toast.Title>
          <Toast.Description>
            This toast will disappear automatically in 3 seconds.
          </Toast.Description>
          <Toast.CloseTrigger>×</Toast.CloseTrigger>
        </Toast.Root>
      )}
    </div>
  );
}
```

### Toast Manager with Queue

```tsx
import { useState, useCallback } from "react";
import { Toast } from "@liquidify/components/toast";

export default function ToastManager() {
  const [toasts, setToasts] = useState([]);

  const addToast = useCallback((toast) => {
    const id = Date.now();
    const newToast = { ...toast, id };
    setToasts(prev => [...prev, newToast]);

    // Auto-remove after 4 seconds
    setTimeout(() => {
      setToasts(prev => prev.filter(t => t.id !== id));
    }, 4000);
  }, []);

  const removeToast = useCallback((id) => {
    setToasts(prev => prev.filter(t => t.id !== id));
  }, []);

  return (
    <div>
      <div style={{ display: "flex", gap: "1rem", marginBottom: "2rem" }}>
        <button onClick={() => addToast({ 
          status: "success", 
          title: "Success!", 
          description: "Task completed successfully." 
        })}>
          Add Success Toast
        </button>
        <button onClick={() => addToast({ 
          status: "error", 
          title: "Error!", 
          description: "An error occurred." 
        })}>
          Add Error Toast
        </button>
        <button onClick={() => addToast({ 
          status: "info", 
          title: "Info", 
          description: "Here's some information." 
        })}>
          Add Info Toast
        </button>
      </div>

      <div style={{ 
        position: "fixed", 
        top: "1rem", 
        right: "1rem", 
        zIndex: 1000,
        display: "flex",
        flexDirection: "column",
        gap: "0.5rem"
      }}>
        {toasts.map((toast) => (
          <Toast.Root 
            key={toast.id} 
            open={true} 
            onOpenChange={() => removeToast(toast.id)}
            status={toast.status}
          >
            <Toast.Title>{toast.title}</Toast.Title>
            <Toast.Description>{toast.description}</Toast.Description>
            <Toast.CloseTrigger onClick={() => removeToast(toast.id)}>
              ×
            </Toast.CloseTrigger>
          </Toast.Root>
        ))}
      </div>
    </div>
  );
}
```

### Loading Toast

```tsx
import { useState } from "react";
import { Toast } from "@liquidify/components/toast";

export default function LoadingToast() {
  const [loadingToast, setLoadingToast] = useState(false);

  const handleAsyncAction = async () => {
    setLoadingToast(true);
    
    try {
      // Simulate async operation
      await new Promise(resolve => setTimeout(resolve, 2000));
      
      setLoadingToast(false);
      // Could show success toast here
    } catch (error) {
      setLoadingToast(false);
      // Could show error toast here
    }
  };

  return (
    <div>
      <button onClick={handleAsyncAction}>
        Start Process
      </button>
      
      {loadingToast && (
        <Toast.Root open={loadingToast} onOpenChange={setLoadingToast}>
          <Toast.Title>Processing...</Toast.Title>
          <Toast.Description>
            Please wait while we process your request.
          </Toast.Description>
        </Toast.Root>
      )}
    </div>
  );
}
```

## API Reference

### Toast.Root

The root container for the toast component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Whether the toast is visible |
| `onOpenChange` | `(open: boolean) => void` | - | Callback fired when visibility changes |
| `status` | `"info" \| "success" \| "warning" \| "error"` | `"info"` | The status/type of the toast |
| `duration` | `number` | `5000` | Auto-dismiss duration in milliseconds |
| `persist` | `boolean` | `false` | Whether the toast should persist until manually closed |
| `pauseOnHover` | `boolean` | `true` | Whether to pause auto-dismiss on hover |
| `pauseOnPageIdle` | `boolean` | `true` | Whether to pause auto-dismiss when page is idle |

### Toast.Title

The title/heading of the toast.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The title content |

### Toast.Description

The description text of the toast.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The description content |

### Toast.CloseTrigger

The button to close/dismiss the toast.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The close trigger content |
| `onClick` | `() => void` | - | Custom click handler |

### Toast.ActionTrigger

An action button within the toast.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The action trigger content |
| `onClick` | `() => void` | - | Action click handler |

## Accessibility

The Toast component includes comprehensive accessibility features:

- **ARIA Live Regions**: Automatically announces toast content to screen readers
- **Focus Management**: Manages focus appropriately when toasts appear/disappear
- **Keyboard Navigation**: Supports keyboard interaction for actions and dismissal
- **High Contrast**: Compatible with high contrast mode
- **Reduced Motion**: Respects user preferences for reduced motion

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Escape` | Close the focused toast |
| `Tab` | Navigate between toast actions |
| `Enter` / `Space` | Activate focused action or close button |

## Best Practices

1. **Appropriate Timing**: Use auto-dismiss for informational toasts, persistent for critical actions
2. **Clear Messaging**: Keep toast content concise and actionable
3. **Status Colors**: Use appropriate status types to convey the right context
4. **Action Buttons**: Provide clear actions when user intervention is needed
5. **Queue Management**: Limit the number of simultaneous toasts to avoid overwhelming users
6. **Positioning**: Place toasts in consistent, non-intrusive locations
7. **Accessibility**: Ensure toasts are announced to screen readers and keyboard accessible
