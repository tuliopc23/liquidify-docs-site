# Card

Flexible container component for grouping related content. Provides visual boundaries and hierarchy for organizing information into digestible sections.

## Import

```tsx
import { Card } from "liquidify-react/card"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<Card.Root>
			<Card.Header>
				<Card.Title>Card Title</Card.Title>
				<Card.Description>
					A brief description of the card content goes here.
				</Card.Description>
			</Card.Header>
			<Card.Body>
				<p>Main content area for your card information.</p>
			</Card.Body>
			<Card.Footer>
				<Button>Action</Button>
			</Card.Footer>
		</Card.Root>
	)
}
```

## Simple Card

```tsx
export default function SimpleExample() {
	return (
		<Card.Root>
			<Card.Body>
				<h3>Simple Card</h3>
				<p>Just content without header or footer.</p>
			</Card.Body>
		</Card.Root>
	)
}
```

## Interactive Card

```tsx
export default function InteractiveExample() {
	return (
		<Card.Root asChild>
			<a href="/article" style={{ textDecoration: "none", color: "inherit" }}>
				<Card.Header>
					<Card.Title>Read Article</Card.Title>
					<Card.Description>Click to read more</Card.Description>
				</Card.Header>
				<Card.Body>
					<p>This entire card is clickable and navigates to the article.</p>
				</Card.Body>
			</a>
		</Card.Root>
	)
}
```

## With Image

```tsx
export default function ImageExample() {
	return (
		<Card.Root>
			<img 
				src="/card-image.jpg" 
				alt="Card image"
				style={{ width: "100%", height: "200px", objectFit: "cover" }}
			/>
			<Card.Header>
				<Card.Title>Image Card</Card.Title>
			</Card.Header>
			<Card.Body>
				<p>Card content with an image at the top.</p>
			</Card.Body>
		</Card.Root>
	)
}
```

## API Reference

### Root Props
- `asChild`: `boolean` - Render as child element (for links/buttons)

### Accessibility
- Proper heading hierarchy with Card.Title
- Semantic structure for screen readers
- Focus management for interactive cards
- High contrast borders and backgrounds
