# Badge

Small status indicators and labels for displaying categories, counts, or states. Perfect for notifications, tags, and highlighting important information.

## Import

```tsx
import { Badge } from "liquidify-react/badge"
```

## Basic Usage

```tsx
export default function Example() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Badge>Default</Badge>
			<Badge variant="secondary">Secondary</Badge>
			<Badge variant="success">Success</Badge>
			<Badge variant="warning">Warning</Badge>
			<Badge variant="destructive">Error</Badge>
		</div>
	)
}
```

## Sizes

```tsx
export default function SizesExample() {
	return (
		<div style={{ display: "flex", gap: "1rem", alignItems: "center" }}>
			<Badge size="sm">Small</Badge>
			<Badge size="md">Medium</Badge>
			<Badge size="lg">Large</Badge>
		</div>
	)
}
```

## With Icons

```tsx
export default function IconExample() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Badge variant="success">
				<CheckIcon />
				Completed
			</Badge>
			
			<Badge variant="warning">
				<AlertIcon />
				Pending
			</Badge>
		</div>
	)
}
```

## Notification Badges

```tsx
export default function NotificationExample() {
	return (
		<div style={{ position: "relative", display: "inline-block" }}>
			<Button>Messages</Button>
			<Badge 
				variant="destructive" 
				size="sm"
				style={{ 
					position: "absolute", 
					top: "-8px", 
					right: "-8px" 
				}}
			>
				3
			</Badge>
		</div>
	)
}
```

## API Reference

### Props
- `variant`: `"default" | "secondary" | "success" | "warning" | "destructive"` - Color variant
- `size`: `"sm" | "md" | "lg"` - Badge size

### Accessibility
- Meaningful text content for screen readers
- Proper color contrast ratios
- Icon descriptions when present
