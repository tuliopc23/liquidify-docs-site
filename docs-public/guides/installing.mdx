# Installing

## Installation Guide

This guide covers installing LiqUIdify in various environments and frameworks. Choose the approach that matches your setup.

## Package Manager Installation

### Basic Installation

Install LiqUIdify with your preferred package manager:

```bash
# Using bun (recommended)
bun add liquidify-react @ark-ui/react framer-motion lucide-react

# Using npm
npm install liquidify-react @ark-ui/react framer-motion lucide-react

# Using pnpm
pnpm add liquidify-react @ark-ui/react framer-motion lucide-react

# Using yarn
yarn add liquidify-react @ark-ui/react framer-motion lucide-react
```

### Latest Versions

For bleeding-edge features, install the latest versions:

```bash
# bun
bun add liquidify-react@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest

# npm
npm install liquidify-react@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest

# yarn
yarn add liquidify-react@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest
```

## Framework-Specific Setup

### Next.js

LiqUIdify works seamlessly with Next.js. Import styles in your root layout or `_app.tsx`:

```tsx
// app/layout.tsx (App Router)
import 'liquidify-react/styles'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

```tsx
// pages/_app.tsx (Pages Router)
import 'liquidify-react/styles'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

### Vite

Import styles in your main entry file:

```tsx
// src/main.tsx
import 'liquidify-react/styles'
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import App from './App'

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>
)
```

### Create React App

Import styles in your index file:

```tsx
// src/index.tsx
import 'liquidify-react/styles'
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import App from './App'

const root = createRoot(document.getElementById('root') as HTMLElement)
root.render(
  <StrictMode>
    <App />
  </StrictMode>
)
```

### Remix

Import styles in your root component:

```tsx
// app/root.tsx
import type { LinksFunction } from '@remix-run/node'
import 'liquidify-react/styles'

export const links: LinksFunction = () => []

export default function App() {
  return (
    <html>
      <head>
        <meta charSet="utf-8" />
        <meta name="viewport" content="width=device-width,initial-scale=1" />
      </head>
      <body>
        <Outlet />
      </body>
    </html>
  )
}
```

### Astro

For Astro projects with React integration:

```tsx
// src/layouts/Layout.astro
---
import 'liquidify-react/styles'
---

<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My App</title>
  </head>
  <body>
    <slot />
  </body>
</html>
```

## Peer Dependencies

LiqUIdify requires these peer dependencies:

### Required
- **React** 18 or 19
- **React DOM** 18 or 19  
- **@ark-ui/react** ^5.23.0
- **framer-motion** ^12
- **lucide-react** ^0.544.0

### Version Compatibility

| LiqUIdify | React | Ark UI | Framer Motion |
|-----------|-------|--------|---------------|
| 0.6.x     | 18-19 | ^5.23  | ^12           |
| 0.5.x     | 18    | ^5.20  | ^11           |

## Troubleshooting

### Common Issues

#### Peer Dependency Warnings
If you see peer dependency warnings, install the missing packages:

```bash
npm install @ark-ui/react framer-motion lucide-react
```

#### CSS Not Loading
Ensure you've imported the styles in your app's entry point:

```tsx
import 'liquidify-react/styles'
```

#### TypeScript Errors
Make sure you have compatible TypeScript and React types:

```bash
npm install -D typescript@latest @types/react@latest @types/react-dom@latest
```

#### Build Errors
For build issues, ensure your bundler supports CSS imports and ES modules.

### Environment Variables

For custom CSS file paths, you can set:

```bash
# Custom CSS path (advanced usage)
LIQUIDIFY_CSS_PATH="./custom-liquidify.css"
```

## Verification

After installation, verify everything works by creating a simple component:

```tsx
import { Button } from 'liquidify-react/button'

function Test() {
  return <Button>Hello LiqUIdify!</Button>
}
```

If the button renders with Apple-inspired styling, you're ready to go!
