# Getting Started

## Quick Setup Guide

Get LiqUIdify running in your React project in under 5 minutes. This guide covers the fastest path to start building with Apple-quality components.

### Prerequisites

Before you begin, ensure you have:

- **Node.js** 16 or later
- A **package manager**: bun, npm, pnpm, or yarn
- **React** 18 or 19
- A **React project** (Next.js, Vite, Create React App, etc.)

### Installation

Install LiqUIdify and its peer dependencies:

```bash
# Using bun (recommended)
bun add liquidify-react @ark-ui/react framer-motion lucide-react

# Using npm
npm install liquidify-react @ark-ui/react framer-motion lucide-react

# Using pnpm
pnpm add liquidify-react @ark-ui/react framer-motion lucide-react

# Using yarn
yarn add liquidify-react @ark-ui/react framer-motion lucide-react
```

### Setup Styles

Import LiqUIdify's CSS in your app's entry point:

```tsx
// In your main.tsx, index.tsx, or _app.tsx
import "liquidify-react/styles";
import { createRoot } from "react-dom/client";
import App from "./App";

const root = createRoot(document.getElementById("root")!);
root.render(<App />);
```

### Your First Component

Start using LiqUIdify components immediately:

```tsx
import { Button } from "liquidify-react/button";
import { Card } from "liquidify-react/card";

function App() {
  return (
    <Card variant="glass" padded>
      <h2>Welcome to LiqUIdify</h2>
      <p>Apple-quality components for your React app</p>
      <div style={{ marginTop: "1rem" }}>
        <Button variant="filled" tone="accent">
          Get Started
        </Button>
      </div>
    </Card>
  );
}
```

### Verify Installation

If everything is set up correctly, you should see a beautifully styled card with liquid glass effects. The button should have smooth hover animations, capsule-shaped borders, and Apple-inspired typography.

## Common Issues

### Styles Not Applied

Ensure you've imported the CSS file in your app's entry point:

```tsx
import "liquidify-react/styles";
```

### TypeScript Errors

Make sure you have the latest version of TypeScript (5.0+) and React types:

```bash
npm install -D typescript @types/react @types/react-dom
```

### Peer Dependency Warnings

Install all required peer dependencies as shown in the installation step above.
