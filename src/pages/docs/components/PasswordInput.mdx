# PasswordInput

A specialized input component for password entry with built-in visibility toggle functionality. Perfect for login forms, registration, password changes, or any scenario requiring secure password input with user-friendly visibility controls.

## Installation

```bash
npm install @liquidify/components
```

## Import

```tsx
import { PasswordInput } from "@liquidify/components/passwordInput";
```

## Usage

### Basic Password Input

```tsx
import { useState } from "react";
import { PasswordInput } from "@liquidify/components/passwordInput";

export default function BasicPasswordInput() {
  const [password, setPassword] = useState("");

  return (
    <div style={{ maxWidth: "300px", margin: "0 auto", padding: "2rem" }}>
      <h3>Enter Password</h3>

      <PasswordInput.Root>
        <PasswordInput.Input
          placeholder="Enter your password"
          value={password}
          onChange={(e) => setPassword(e.target.value)}
        />
        <PasswordInput.VisibilityTrigger />
      </PasswordInput.Root>

      <p style={{ marginTop: "1rem", fontSize: "0.9rem", color: "#666" }}>
        Password length: {password.length} characters
      </p>
    </div>
  );
}
```

### Registration Form with Validation

```tsx
import { useState } from "react";
import { PasswordInput } from "@liquidify/components/passwordInput";

export default function RegistrationForm() {
  const [formData, setFormData] = useState({
    username: "",
    email: "",
    password: "",
    confirmPassword: ""
  });
  const [errors, setErrors] = useState({});
  const [passwordStrength, setPasswordStrength] = useState(0);

  const validatePassword = (password) => {
    let strength = 0;
    const checks = [
      password.length >= 8,
      /[A-Z]/.test(password),
      /[a-z]/.test(password),
      /[0-9]/.test(password),
      /[^A-Za-z0-9]/.test(password)
    ];

    strength = checks.filter(Boolean).length;
    return strength;
  };

  const getPasswordStrengthText = (strength) => {
    switch (strength) {
      case 0: return "Very Weak";
      case 1: return "Weak";
      case 2: return "Fair";
      case 3: return "Good";
      case 4: return "Strong";
      case 5: return "Very Strong";
      default: return "Very Weak";
    }
  };

  const getPasswordStrengthColor = (strength) => {
    switch (strength) {
      case 0: case 1: return "#dc3545";
      case 2: return "#ffc107";
      case 3: return "#17a2b8";
      case 4: case 5: return "#28a745";
      default: return "#dc3545";
    }
  };

  const validateForm = () => {
    const newErrors = {};

    if (!formData.username) newErrors.username = "Username is required";
    if (!formData.email) newErrors.email = "Email is required";
    else if (!/\S+@\S+\.\S+/.test(formData.email)) newErrors.email = "Email is invalid";

    if (!formData.password) newErrors.password = "Password is required";
    else if (formData.password.length < 8) newErrors.password = "Password must be at least 8 characters";

    if (!formData.confirmPassword) newErrors.confirmPassword = "Please confirm your password";
    else if (formData.password !== formData.confirmPassword) newErrors.confirmPassword = "Passwords do not match";

    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  const handleSubmit = (e) => {
    e.preventDefault();
    if (validateForm()) {
      console.log("Registration data:", formData);
      alert("Registration successful!");
    }
  };

  const updateFormData = (field, value) => {
    setFormData(prev => ({ ...prev, [field]: value }));

    if (field === "password") {
      setPasswordStrength(validatePassword(value));
    }

    // Clear errors when user starts typing
    if (errors[field]) {
      setErrors(prev => ({ ...prev, [field]: "" }));
    }
  };

  return (
    <div style={{ maxWidth: "400px", margin: "0 auto", padding: "2rem" }}>
      <h3>Create Account</h3>

      <form onSubmit={handleSubmit}>
        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.5rem" }}>
            Username *
          </label>
          <input
            type="text"
            value={formData.username}
            onChange={(e) => updateFormData("username", e.target.value)}
            placeholder="Choose a username"
            style={{
              width: "100%",
              padding: "0.5rem",
              border: errors.username ? "1px solid #dc3545" : "1px solid #ccc",
              borderRadius: "4px"
            }}
          />
          {errors.username && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>
              {errors.username}
            </div>
          )}
        </div>

        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.5rem" }}>
            Email Address *
          </label>
          <input
            type="email"
            value={formData.email}
            onChange={(e) => updateFormData("email", e.target.value)}
            placeholder="Enter your email"
            style={{
              width: "100%",
              padding: "0.5rem",
              border: errors.email ? "1px solid #dc3545" : "1px solid #ccc",
              borderRadius: "4px"
            }}
          />
          {errors.email && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>
              {errors.email}
            </div>
          )}
        </div>

        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.5rem" }}>
            Password *
          </label>
          <PasswordInput.Root>
            <PasswordInput.Input
              value={formData.password}
              onChange={(e) => updateFormData("password", e.target.value)}
              placeholder="Create a strong password"
              style={{
                border: errors.password ? "1px solid #dc3545" : "1px solid #ccc"
              }}
            />
            <PasswordInput.VisibilityTrigger />
          </PasswordInput.Root>

          {formData.password && (
            <div style={{ marginTop: "0.5rem" }}>
              <div style={{ display: "flex", justifyContent: "space-between", marginBottom: "0.25rem" }}>
                <span style={{ fontSize: "0.9rem" }}>Password Strength:</span>
                <span style={{
                  fontSize: "0.9rem",
                  color: getPasswordStrengthColor(passwordStrength)
                }}>
                  {getPasswordStrengthText(passwordStrength)}
                </span>
              </div>
              <div style={{
                width: "100%",
                height: "4px",
                backgroundColor: "#e9ecef",
                borderRadius: "2px"
              }}>
                <div style={{
                  width: `${(passwordStrength / 5) * 100}%`,
                  height: "100%",
                  backgroundColor: getPasswordStrengthColor(passwordStrength),
                  borderRadius: "2px",
                  transition: "width 0.3s ease"
                }} />
              </div>
            </div>
          )}

          {errors.password && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>
              {errors.password}
            </div>
          )}
        </div>

        <div style={{ marginBottom: "1.5rem" }}>
          <label style={{ display: "block", marginBottom: "0.5rem" }}>
            Confirm Password *
          </label>
          <PasswordInput.Root>
            <PasswordInput.Input
              value={formData.confirmPassword}
              onChange={(e) => updateFormData("confirmPassword", e.target.value)}
              placeholder="Confirm your password"
              style={{
                border: errors.confirmPassword ? "1px solid #dc3545" : "1px solid #ccc"
              }}
            />
            <PasswordInput.VisibilityTrigger />
          </PasswordInput.Root>

          {errors.confirmPassword && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>
              {errors.confirmPassword}
            </div>
          )}
        </div>

        <button
          type="submit"
          style={{
            width: "100%",
            padding: "0.75rem",
            backgroundColor: "#28a745",
            color: "white",
            border: "none",
            borderRadius: "4px",
            cursor: "pointer"
          }}
        >
          Create Account
        </button>
      </form>
    </div>
  );
}
```

```tsx
import { useState } from "react";
import { PasswordInput } from "@liquidify/components/passwordInput";

export default function PasswordChangeForm() {
  const [formData, setFormData] = useState({
    currentPassword: "",
    newPassword: "",
    confirmNewPassword: ""
  });
  const [errors, setErrors] = useState({});
  const [isSubmitting, setIsSubmitting] = useState(false);

  const validateForm = () => {
    const newErrors = {};

    if (!formData.currentPassword) {
      newErrors.currentPassword = "Current password is required";
    }

    if (!formData.newPassword) {
      newErrors.newPassword = "New password is required";
    } else if (formData.newPassword.length < 8) {
      newErrors.newPassword = "New password must be at least 8 characters";
    } else if (formData.newPassword === formData.currentPassword) {
      newErrors.newPassword = "New password must be different from current password";
    }

    if (!formData.confirmNewPassword) {
      newErrors.confirmNewPassword = "Please confirm your new password";
    } else if (formData.newPassword !== formData.confirmNewPassword) {
      newErrors.confirmNewPassword = "Passwords do not match";
    }

    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  const handleSubmit = async (e) => {
    e.preventDefault();

    if (!validateForm()) return;

    setIsSubmitting(true);

    // Simulate API call
    await new Promise(resolve => setTimeout(resolve, 2000));

    console.log("Password change:", {
      currentPassword: formData.currentPassword,
      newPassword: formData.newPassword
    });

    setIsSubmitting(false);
    alert("Password changed successfully!");

    // Reset form
    setFormData({
      currentPassword: "",
      newPassword: "",
      confirmNewPassword: ""
    });
  };

  const updateFormData = (field, value) => {
    setFormData(prev => ({ ...prev, [field]: value }));

    // Clear errors when user starts typing
    if (errors[field]) {
      setErrors(prev => ({ ...prev, [field]: "" }));

  return (

              placeholder="Enter current password"
              style={{
                border: errors.currentPassword ? "1px solid #dc3545" : "1px solid #ccc"
              }}
            />
            <PasswordInput.VisibilityTrigger />
          </PasswordInput.Root>

          {errors.currentPassword && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>
              {errors.currentPassword}
            </div>
          )}
        </div>

        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.5rem" }}>
            New Password *
          </label>
          <PasswordInput.Root>
            <PasswordInput.Input
              value={formData.newPassword}
              onChange={(e) => updateFormData("newPassword", e.target.value)}
              placeholder="Enter new password"
              style={{
                border: errors.newPassword ? "1px solid #dc3545" : "1px solid #ccc"
              }}
            />
            <PasswordInput.VisibilityTrigger />
          </PasswordInput.Root>

          {errors.newPassword && (
            <div style={{ color: "#dc3545", fontSize: "0.9rem", marginTop: "0.25rem" }}>

        </div>
```

```tsx
import { useState, useEffect } from "react";
import { PasswordInput } from "@liquidify/components/passwordInput";

export default function PasswordRequirements() {
  const [password, setPassword] = useState("");
  const [requirements, setRequirements] = useState({
    length: false,
    uppercase: false,
    lowercase: false,
    number: false,
    special: false
  });

  useEffect(() => {
    setRequirements({
      length: password.length >= 8,
      uppercase: /[A-Z]/.test(password),
      lowercase: /[a-z]/.test(password),
      number: /[0-9]/.test(password),
      special: /[^A-Za-z0-9]/.test(password)
    });
  }, [password]);

  const allRequirementsMet = Object.values(requirements).every(Boolean);

  const requirementItems = [
    { key: "length", text: "At least 8 characters" },
    { key: "uppercase", text: "One uppercase letter" },
    { key: "lowercase", text: "One lowercase letter" },
    { key: "number", text: "One number" },
    { key: "special", text: "One special character" }

  return (
    <div style={{ maxWidth: "400px", margin: "0 auto", padding: "2rem" }}>
      <h3>Create Secure Password</h3>

      <div style={{ marginBottom: "1rem" }}>
        <PasswordInput.Root>
          <PasswordInput.Input
            value={password}
            onChange={(e) => setPassword(e.target.value)}
            placeholder="Enter a strong password"
          />
          <PasswordInput.VisibilityTrigger />
        </PasswordInput.Root>
      </div>

      <div style={{ marginBottom: "1rem" }}>
        <h4>Password Requirements:</h4>
        <div style={{ display: "flex", flexDirection: "column", gap: "0.5rem" }}>
          {requirementItems.map(item => (
            <div
              key={item.key}
              style={{

                color: requirements[item.key] ? "#28a745" : "#666"
              }}
            >
              <span style={{ fontSize: "1.2rem" }}>
                {requirements[item.key] ? "✓" : "○"}
              </span>
              <span style={{
                textDecoration: requirements[item.key] ? "line-through" : "none"
              }}>
                {item.text}
              </span>
            </div>
          ))}
        </div>
      </div>

      {password && (
        <div style={{
          padding: "1rem",
          borderRadius: "8px",

          border: `1px solid ${allRequirementsMet ? "#c3e6cb" : "#ffeaa7"}`
        }}>
          <div style={{
            color: allRequirementsMet ? "#155724" : "#856404",

          }}>
            {allRequirementsMet ? "✓ Strong Password" : "⚠ Password Too Weak"}
          </div>
          <div style={{
            color: allRequirementsMet ? "#155724" : "#856404",
            marginTop: "0.25rem"

            {allRequirementsMet
              ? "Your password meets all security requirements."
              : "Please ensure your password meets all the requirements above."
            }
          </div>
        </div>
      )}
    </div>
  );
}
```

## API Reference

### PasswordInput.Root

The root container for the password input component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The password input content |
| All standard div props | - | - | Standard HTML div attributes |

### PasswordInput.Input

The password input field.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text for the input |
| `value` | `string` | - | Controlled input value |
| `onChange` | `(event: ChangeEvent) => void` | - | Change event handler |
| `disabled` | `boolean` | `false` | Whether the input is disabled |
| `required` | `boolean` | `false` | Whether the input is required |
| All standard input props | - | - | Standard HTML input attributes |

### PasswordInput.VisibilityTrigger

The button to toggle password visibility.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Custom trigger content |
| All standard button props | - | - | Standard HTML button attributes |

## Accessibility

The PasswordInput component includes comprehensive accessibility features:

- **ARIA Labels**: Proper labeling for screen readers
- **Keyboard Navigation**: Full keyboard support for toggling visibility
- **Screen Reader Support**: Announces password visibility state changes
- **Focus Management**: Proper focus handling and visual indicators
- **High Contrast**: Compatible with high contrast themes

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Tab` | Navigate to visibility trigger |
| `Enter` / `Space` | Toggle password visibility |
| `Escape` | Move focus away from trigger |

## Best Practices

1. **Security First**: Never store passwords in plain text
2. **Clear Labels**: Always provide clear labels for password fields
3. **Validation**: Implement client-side and server-side password validation
4. **Strength Indicators**: Show password strength feedback when creating passwords
5. **Confirmation**: Require password confirmation for important changes
6. **Visibility Toggle**: Always include the visibility toggle for better UX
7. **Requirements**: Display password requirements clearly
8. **Error Handling**: Provide clear error messages for validation failures
