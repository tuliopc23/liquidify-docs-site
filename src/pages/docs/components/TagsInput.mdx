# TagsInput

An input component that allows users to enter, edit, and delete multiple tags. Perfect for categorization, labeling, and creating lists of related items with an intuitive interface.

## Installation

```bash
npm install liquidify-react
```

## Import

```tsx
import { TagsInput } from "liquidify-react/tagsInput";
```

## Usage

### Basic TagsInput

```tsx
import { TagsInput } from "liquidify-react/tagsInput";

export default function BasicTagsInput() {
  return (
    <TagsInput.Root>
      <TagsInput.Label>Skills</TagsInput.Label>
      <TagsInput.Control>
        {["React", "TypeScript"].map((value, index) => (
          <TagsInput.Item key={index} index={index} value={value}>
            <TagsInput.ItemPreview>
              <TagsInput.ItemText>{value}</TagsInput.ItemText>
              <TagsInput.ItemDeleteTrigger>×</TagsInput.ItemDeleteTrigger>
            </TagsInput.ItemPreview>
            <TagsInput.ItemInput />
          </TagsInput.Item>
        ))}
        <TagsInput.Input placeholder="Add a skill..." />
        <TagsInput.ClearTrigger>Clear all</TagsInput.ClearTrigger>
      </TagsInput.Control>
      <TagsInput.HiddenInput />
    </TagsInput.Root>
  );
}
```

### TagsInput with Validation

```tsx
import { useState } from "react";
import { TagsInput } from "liquidify-react/tagsInput";

export default function ValidatedTagsInput() {
  const [tags, setTags] = useState(["email", "urgent"]);
  const [error, setError] = useState("");

  const handleValueChange = (newTags: string[]) => {
    if (newTags.length > 5) {
      setError("Maximum 5 tags allowed");
      return;
    }
    if (newTags.some(tag => tag.length > 20)) {
      setError("Tags must be 20 characters or less");
      return;
    }
    setError("");
    setTags(newTags);
  };

  return (
    <TagsInput.Root value={tags} onValueChange={handleValueChange}>
      <TagsInput.Label>Email Labels ({tags.length}/5)</TagsInput.Label>
      <TagsInput.Control>
        {tags.map((value, index) => (
          <TagsInput.Item key={index} index={index} value={value}>
            <TagsInput.ItemPreview>
              <TagsInput.ItemText>{value}</TagsInput.ItemText>
              <TagsInput.ItemDeleteTrigger>×</TagsInput.ItemDeleteTrigger>
            </TagsInput.ItemPreview>
            <TagsInput.ItemInput />
          </TagsInput.Item>
        ))}
        <TagsInput.Input 
          placeholder="Add label..." 
          disabled={tags.length >= 5}
        />
        {tags.length > 0 && (
          <TagsInput.ClearTrigger>Clear all</TagsInput.ClearTrigger>
        )}
      </TagsInput.Control>
      <TagsInput.HiddenInput />
      {error && <p style={{ color: "red", fontSize: "14px" }}>{error}</p>}
    </TagsInput.Root>
  );
}
```

### Custom Delimiters

```tsx
import { useState } from "react";
import { TagsInput } from "liquidify-react/tagsInput";

export default function CustomDelimitersTagsInput() {
  const [tags, setTags] = useState(["apple", "banana"]);

  return (
    <TagsInput.Root 
      value={tags} 
      onValueChange={setTags}
      delimiter={[",", ";", "|"]}
    >
      <TagsInput.Label>Fruits (separate with comma, semicolon, or pipe)</TagsInput.Label>
      <TagsInput.Control>
        {tags.map((value, index) => (
          <TagsInput.Item key={index} index={index} value={value}>
            <TagsInput.ItemPreview>
              <TagsInput.ItemText>{value}</TagsInput.ItemText>
              <TagsInput.ItemDeleteTrigger>×</TagsInput.ItemDeleteTrigger>
            </TagsInput.ItemPreview>
            <TagsInput.ItemInput />
          </TagsInput.Item>
        ))}
        <TagsInput.Input placeholder="apple, banana; orange | grape" />
        <TagsInput.ClearTrigger>Clear</TagsInput.ClearTrigger>
      </TagsInput.Control>
      <TagsInput.HiddenInput />
    </TagsInput.Root>
  );
}
```

## API Reference

### TagsInput.Root

The root container for the tags input component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | - | The controlled values of the tags input |
| `defaultValue` | `string[]` | `[]` | The default values when uncontrolled |
| `onValueChange` | `(values: string[]) => void` | - | Callback fired when the values change |
| `disabled` | `boolean` | `false` | Whether the tags input is disabled |
| `readOnly` | `boolean` | `false` | Whether the tags input is read-only |
| `editable` | `boolean` | `false` | Whether tags can be edited inline |
| `delimiter` | `string \| string[]` | `","` | Characters that trigger tag creation |
| `max` | `number` | - | Maximum number of tags allowed |
| `allowOverflow` | `boolean` | `false` | Whether to allow exceeding the max |
| `validate` | `(value: string) => boolean` | - | Custom validation function |

### TagsInput.Label

The label for the tags input.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The label content |

### TagsInput.Control

The container for tags and input.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The control content |

### TagsInput.Input

The input field for adding new tags.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text for the input |

### TagsInput.Item

A tag item container.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | - | The index of the tag |
| `value` | `string` | - | The value of the tag |
| `children` | `ReactNode` | - | The tag content |

### TagsInput.ItemPreview

## Accessibility

The TagsInput component includes comprehensive accessibility features:

- **Keyboard Navigation**: Full keyboard support for managing tags
- **Focus Management**: Proper focus handling between tags and input
- **ARIA Labels**: Implements proper ARIA roles and labels
- **Screen Reader Support**: Announces tag changes and actions
- **High Contrast**: Supports high contrast mode for better visibility

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Enter` | Create a new tag from input value |
| `Backspace` | Delete the last tag when input is empty |
| `Delete` | Delete the focused tag |
| `Arrow Left/Right` | Navigate between tags |
| `Tab` | Move focus to next element |
| `Shift + Tab` | Move focus to previous element |
| `Escape` | Cancel tag editing or clear input |
| `F2` | Start editing the focused tag (when editable) |

## Best Practices

1. **Clear Labels**: Use descriptive labels that explain the purpose of the tags
2. **Validation**: Implement appropriate validation for tag content and limits
3. **Visual Feedback**: Provide clear visual feedback for invalid or duplicate tags
4. **Delimiter Choice**: Choose delimiters that make sense for your use case
5. **Performance**: Consider virtualization for large numbers of tags
6. **Accessibility**: Ensure proper keyboard navigation and screen reader support
7. **Form Integration**: Use the hidden input for proper form submission
