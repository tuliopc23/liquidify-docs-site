# PinInput

A PIN or OTP input component that splits values across multiple input fields for secure code entry.

## Installation

```bash
bun add liquidify-react
```

## Import

```tsx
import { PinInput } from "liquidify-react/ark-ui/pinInput";
```

## Basic Usage

Create a basic PIN input for verification codes:

```tsx
import { PinInput } from "liquidify-react/ark-ui/pinInput";

export function BasicExample() {
  return (
    <PinInput.Root>
      <PinInput.Label>Enter verification code</PinInput.Label>
      <PinInput.Control>
        <PinInput.Input index={0} />
        <PinInput.Input index={1} />
        <PinInput.Input index={2} />
        <PinInput.Input index={3} />
      </PinInput.Control>
    </PinInput.Root>
  );
}
```

## Controlled Value

Control the PIN input value with state:

```tsx
import { useState } from "react";
import { PinInput } from "liquidify-react/ark-ui/pinInput";

export function ControlledExample() {
  const [value, setValue] = useState(["", "", "", ""]);

  return (
    <PinInput.Root value={value} onValueChange={(details) => setValue(details.value)}>
      <PinInput.Label>Enter your PIN</PinInput.Label>
      <PinInput.Control>
        <PinInput.Input index={0} />
        <PinInput.Input index={1} />
        <PinInput.Input index={2} />
        <PinInput.Input index={3} />
      </PinInput.Control>
      <p>Current value: {value.join("")}</p>
    </PinInput.Root>
  );
}
```

## Validation and Complete Handler

Handle completion and validation:

```tsx
import { useState } from "react";
import { PinInput } from "liquidify-react/ark-ui/pinInput";

export function ValidationExample() {
  const [isValid, setIsValid] = useState(null);

  const handleComplete = (details) => {
    const code = details.value.join("");
    const valid = code === "1234";
    setIsValid(valid);

    if (valid) {
      console.log("PIN verified successfully!");
    }
  };

  return (
    <PinInput.Root onValueComplete={handleComplete} invalid={isValid === false}>
      <PinInput.Label>Enter PIN (hint: 1234)</PinInput.Label>
      <PinInput.Control>
        <PinInput.Input index={0} />
        <PinInput.Input index={1} />
        <PinInput.Input index={2} />
        <PinInput.Input index={3} />
      </PinInput.Control>
      {isValid === false && (
        <PinInput.ErrorText>Incorrect PIN. Please try again.</PinInput.ErrorText>
      )}
      {isValid === true && (
        <PinInput.HelperText style={{ color: "green" }}>
          PIN verified successfully!
        </PinInput.HelperText>
      )}
    </PinInput.Root>
  );
}
```

## API Reference

### PinInput.Root

The root container for the PIN input.

| Prop              | Type                                          | Default     | Description                         |
| ----------------- | --------------------------------------------- | ----------- | ----------------------------------- |
| `value`           | `string[]`                                    | -           | Controlled value array              |
| `defaultValue`    | `string[]`                                    | -           | Default value array                 |
| `onValueChange`   | `(details) => void`                           | -           | Callback when value changes         |
| `onValueComplete` | `(details) => void`                           | -           | Callback when all inputs are filled |
| `type`            | `"numeric" \| "alphabetic" \| "alphanumeric"` | `"numeric"` | Input type constraint               |
| `mask`            | `boolean`                                     | `false`     | Whether to mask input values        |
| `placeholder`     | `string`                                      | -           | Placeholder for empty inputs        |
| `disabled`        | `boolean`                                     | `false`     | Whether inputs are disabled         |
| `invalid`         | `boolean`                                     | `false`     | Whether inputs are invalid          |
| `autoFocus`       | `boolean`                                     | `false`     | Auto-focus first input on mount     |

### PinInput.Label

The label for the PIN input.

### PinInput.Control

Container for the input fields.

### PinInput.Input

Individual PIN input field.

| Prop    | Type     | Default | Description                         |
| ------- | -------- | ------- | ----------------------------------- |
| `index` | `number` | -       | Index of the input field (required) |

### PinInput.HelperText

Helper text displayed below the inputs.

### PinInput.ErrorText

Error message displayed when inputs are invalid.

## Accessibility

The PinInput component follows WAI-ARIA guidelines:

- Uses `role="group"` for the input container
- Associates labels with `aria-labelledby`
- Implements proper focus management between inputs
- Supports keyboard navigation with arrow keys and backspace
- Provides screen reader announcements for value changes
- Associates error messages with `aria-describedby`

## Best Practices

- Use clear labels indicating the expected code type and length
- Provide helpful context about where the code comes from
- Implement proper validation with clear error messages
- Consider auto-advancing to the next field on input
- Use masking for sensitive PINs but not for verification codes
- Provide option to paste complete codes for better UX
- Test with screen readers and keyboard navigation
