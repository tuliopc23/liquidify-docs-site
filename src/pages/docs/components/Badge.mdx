# Badge

Small status indicators and labels for displaying categories, counts, or states. Perfect for notifications, tags, and highlighting important information.

## Import

```tsx
import { Badge } from "liquidify-react/badge"
```

## With Custom Styling

```tsx
export default function CustomExample() {
	return (
		<div style={{ display: "flex", gap: "1rem" }}>
			<Badge style={{ backgroundColor: '#10b981', color: 'white' }}>
				Success
			</Badge>

			<Badge style={{ backgroundColor: '#f59e0b', color: 'white' }}>
				Warning
			</Badge>
		</div>
	)
}
```

## API Reference

### Props
- `tone`: `"neutral" | "blue"` - Visual tone of the badge (optional)
- Standard HTML span attributes are supported

### Accessibility
- Meaningful text content for screen readers
- Proper color contrast ratios maintained
- Inherits standard span semantics
