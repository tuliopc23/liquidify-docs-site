# QrCode

A QR code generator component that creates scannable codes for encoding text, URLs, contact information, and other data. Perfect for sharing links, contact details, WiFi credentials, or any scenario requiring quick information transfer via mobile devices.

## Installation

```bash
npm install liquidify-react
```

## Import

```tsx
import { QrCode } from "liquidify-react/ark-ui/qrCode";
```

## Usage

### Basic QR Code

```tsx
import { QrCode } from "liquidify-react/ark-ui/qrCode";

export default function BasicQrCode() {
  return (
    <div style={{ textAlign: "center", padding: "2rem" }}>
      <h3>Scan this QR code</h3>
      <QrCode value="https://example.com" />
      <p style={{ marginTop: "1rem", color: "#666" }}>
        Point your camera at the QR code to visit the website
      </p>
    </div>
  );
}
```

### URL Sharing

```tsx
import { useState } from "react";
import { QrCode } from "liquidify-react/ark-ui/qrCode";

export default function UrlSharing() {
  const [url, setUrl] = useState("https://github.com/yourusername/project");

  const predefinedUrls = [
    { label: "Portfolio", url: "https://yourportfolio.com" },
    { label: "GitHub", url: "https://github.com/yourusername" },
    { label: "LinkedIn", url: "https://linkedin.com/in/yourprofile" },
    { label: "Documentation", url: "https://docs.yourproject.com" },
  ];

  return (
    <div style={{ maxWidth: "500px", margin: "0 auto", padding: "2rem" }}>
      <h3>Share URLs via QR Code</h3>

      <div style={{ marginBottom: "2rem" }}>
        <label style={{ display: "block", marginBottom: "0.5rem" }}>Enter URL to encode:</label>
        <input
          type="url"
          value={url}
          onChange={(e) => setUrl(e.target.value)}
          placeholder="https://example.com"
          style={{
            width: "100%",
            padding: "0.5rem",
            border: "1px solid #ccc",
            borderRadius: "4px",
            fontSize: "16px",
          }}
        />
      </div>

      <div style={{ marginBottom: "2rem" }}>
        <h4>Quick Links:</h4>
        <div style={{ display: "flex", flexWrap: "wrap", gap: "0.5rem" }}>
          {predefinedUrls.map((link) => (
            <button
              key={link.label}
              onClick={() => setUrl(link.url)}
              style={{
                padding: "0.5rem 1rem",
                backgroundColor: "#f8f9fa",
                border: "1px solid #dee2e6",
                borderRadius: "4px",
                cursor: "pointer",
              }}
            >
              {link.label}
            </button>
          ))}
        </div>
      </div>

      <div style={{ textAlign: "center" }}>
        <QrCode value={url} />
        <p style={{ marginTop: "1rem", fontSize: "0.9rem", color: "#666" }}>URL: {url}</p>
      </div>
    </div>
  );
}
```

```tsx
import { useState } from "react";

    phone: "+1-555-0123",

VERSION:3.0
FN:${contact.name}
TEL:${contact.phone}
```

### Text Message Encoding

```tsx
import { useState } from "react";
import { QrCode } from "liquidify-react/ark-ui/qrCode";

export default function TextMessageQrCode() {
  const [message, setMessage] = useState("Hello! This is a QR code encoded message.");
  const [phone, setPhone] = useState("");

  // Generate SMS format: smsto:+1234567890:Hello World
  const generateSmsString = (phone, message) => {
    if (phone) {
      return `smsto:${phone}:${message}`;
    }
    return message; // Just encode as plain text if no phone number
  };

  const smsString = generateSmsString(phone, message);

  return (
    <div style={{ maxWidth: "600px", margin: "0 auto", padding: "2rem" }}>
      <h3>Text Message QR Code</h3>
      <p>Create QR codes for text messages or plain text</p>

      <div style={{ marginBottom: "2rem" }}>
        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.25rem" }}>
            Phone Number (optional)
          </label>
          <input
            type="tel"
            value={phone}
            onChange={(e) => setPhone(e.target.value)}
            placeholder="+1-555-0123"
            style={{
              width: "100%",
              padding: "0.5rem",
              border: "1px solid #ccc",
              borderRadius: "4px",
            }}
          />
          <small style={{ color: "#666" }}>
            Include phone number to create a clickable SMS link
          </small>
        </div>

        <div style={{ marginBottom: "1rem" }}>
          <label style={{ display: "block", marginBottom: "0.25rem" }}>Message</label>

              üì± SMS to {phone}
```

## API Reference

### QrCode

The QR code generator component.

#### Props

| Prop                   | Type                       | Default     | Description                                          |
| ---------------------- | -------------------------- | ----------- | ---------------------------------------------------- |
| `value`                | `string`                   | -           | The data to encode in the QR code                    |
| `size`                 | `number`                   | `256`       | The size of the QR code in pixels                    |
| `level`                | `"L" \| "M" \| "Q" \| "H"` | `"M"`       | Error correction level (Low, Medium, Quartile, High) |
| `includeMargin`        | `boolean`                  | `false`     | Whether to include a margin around the QR code       |
| `bgColor`              | `string`                   | `"#FFFFFF"` | Background color                                     |
| `fgColor`              | `string`                   | `"#000000"` | Foreground color                                     |
| All standard div props | -                          | -           | Standard HTML div attributes                         |

## Accessibility

The QrCode component includes comprehensive accessibility features:

- **Alt Text**: Automatically generates descriptive alt text for screen readers
- **ARIA Labels**: Proper labeling for assistive technologies
- **Keyboard Navigation**: Full keyboard support
- **Screen Reader Support**: Announces QR code content and purpose
- **High Contrast**: Compatible with high contrast themes

## Best Practices

1. **Clear Purpose**: Always provide context about what the QR code contains
2. **Error Correction**: Use appropriate error correction levels based on use case
3. **Size Considerations**: Ensure QR codes are large enough to scan reliably
4. **Testing**: Test QR codes with multiple devices and scanning apps
5. **Privacy**: Be mindful of what data you're encoding in QR codes
6. **Updates**: Keep QR code destinations current and functional
7. **Multiple Formats**: Consider providing both QR codes and traditional links
8. **Visual Design**: Use appropriate styling that doesn't interfere with scanning
