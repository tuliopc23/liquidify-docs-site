# Slider

An interactive slider component for selecting numerical values within a specified range.

## Installation

```bash
bun add @liquidify/components
```

## Import

```tsx
import { Slider } from "@liquidify/components/slider"
```

## Basic Usage

Create a simple slider for value selection:

```tsx
import { Slider } from "@liquidify/components/slider"

export function BasicExample() {
  return (
    <Slider.Root defaultValue={[50]} min={0} max={100}>
      <Slider.Label>Volume</Slider.Label>
      <Slider.ValueText />
      <Slider.Control>
        <Slider.Track>
          <Slider.Range />
        </Slider.Track>
        <Slider.Thumb index={0} />
      </Slider.Control>
    </Slider.Root>
  )
}
```

## Controlled Slider

Control the slider value with state:

```tsx
import { useState } from "react"
import { Slider } from "@liquidify/components/slider"

export function ControlledExample() {
  const [value, setValue] = useState([25])

  return (
    <Slider.Root 
      value={value} 
      onValueChange={(details) => setValue(details.value)}
      min={0} 
      max={100}
    >
      <Slider.Label>Price Range ($)</Slider.Label>
      <Slider.ValueText />
      <Slider.Control>
        <Slider.Track>
          <Slider.Range />
        </Slider.Track>
        <Slider.Thumb index={0} />
      </Slider.Control>
      <p>Selected value: ${value[0]}</p>
    </Slider.Root>
  )
}
```

## Range Slider

Create a dual-thumb slider for selecting ranges:

```tsx

  const [range, setRange] = useState([20, 80])

  return (
    <Slider.Root 
      value={range} 
      onValueChange={(details) => setRange(details.value)}
      min={0} 
      max={100}
    >

        <Slider.Label>Price Range</Slider.Label>
        <div style={{ fontSize: "14px", color: "#6b7280" }}>
          ${range[0]} - ${range[1]}
        </div>
      </div>
      <Slider.Control>
        <Slider.Track>
          <Slider.Range />
        </Slider.Track>
        <Slider.Thumb index={0} />
        <Slider.Thumb index={1} />
      </Slider.Control>
    </Slider.Root>
  )
}
```

## Custom Formatting

Format slider values with custom display:

```tsx
import { useState } from "react"
import { Slider } from "@liquidify/components/slider"

export function FormattedExample() {
  const [value, setValue] = useState([50])
  
  const formatValue = (val) => {
    if (val < 25) return "Low"
    if (val < 75) return "Medium"
    return "High"
  }

  return (
    <Slider.Root 
      value={value} 
      onValueChange={(details) => setValue(details.value)}
      min={0} 
      max={100}
    >
      <div style={{ display: "flex", justifyContent: "space-between", alignItems: "center" }}>
        <Slider.Label>Intensity Level</Slider.Label>
        <div style={{ fontWeight: "600" }}>
          {formatValue(value[0])} ({value[0]}%)
        </div>
      </div>
      <Slider.Control>
        <Slider.Track>
          <Slider.Range />
        </Slider.Track>
        <Slider.Thumb index={0} />
      </Slider.Control>
      <div style={{ display: "flex", justifyContent: "space-between", marginTop: "8px", fontSize: "12px", color: "#6b7280" }}>
        <span>Low</span>
        <span>Medium</span>
        <span>High</span>
      </div>
    </Slider.Root>
  )
}
```

## API Reference

### Slider.Root

The root container for the slider component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number[]` | - | Controlled slider values |
| `defaultValue` | `number[]` | `[0]` | Default slider values |
| `onValueChange` | `(details) => void` | - | Callback when value changes |
| `min` | `number` | `0` | Minimum slider value |
| `max` | `number` | `100` | Maximum slider value |
| `step` | `number` | `1` | Step increment for values |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Slider orientation |
| `disabled` | `boolean` | `false` | Whether the slider is disabled |
| `invalid` | `boolean` | `false` | Whether the slider is invalid |

### Slider.Label

The label for the slider component.

### Slider.ValueText

Displays the current slider value(s).

### Slider.Control

Container for the track, range, and thumbs.

### Slider.Track

The background track of the slider.

### Slider.Range

The filled portion indicating the selected range.

### Slider.Thumb

Individual draggable thumb for value selection.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | - | Index of the thumb (required) |

## Accessibility

The Slider component follows WAI-ARIA guidelines:

- Uses `role="slider"` for each thumb
- Implements `aria-valuemin`, `aria-valuemax`, and `aria-valuenow`
- Supports keyboard navigation with arrow keys
- Provides proper focus management
- Associates labels with `aria-labelledby`
- Supports Home/End keys for min/max values
- Implements Page Up/Down for larger increments

## Best Practices

- Provide clear labels indicating what the slider controls
- Use appropriate min/max values and step increments
- Show current values to provide immediate feedback
- Consider using range sliders for selecting ranges
- Ensure sufficient contrast for focus and active states
- Test keyboard navigation and screen reader compatibility
- Use consistent styling across your application
- Consider touch targets for mobile interfaces
