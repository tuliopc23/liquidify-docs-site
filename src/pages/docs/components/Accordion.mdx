# Accordion

Collapsible content sections that expand and collapse to reveal or hide information. Perfect for FAQs, settings panels, and organizing large amounts of content.

## Import

```tsx
import { Accordion } from "liquidify-react/ark-ui/accordion";
```

## Multiple Items

```tsx
<Accordion.Root type="multiple">
  <Accordion.Item value="features">
    <Accordion.ItemTrigger>Features</Accordion.ItemTrigger>
    <Accordion.ItemContent>Accessible, performant, and beautiful components.</Accordion.ItemContent>
  </Accordion.Item>
  <Accordion.Item value="support">
    <Accordion.ItemTrigger>Support</Accordion.ItemTrigger>
    <Accordion.ItemContent>
      Full TypeScript support and comprehensive documentation.
    </Accordion.ItemContent>
  </Accordion.Item>
</Accordion.Root>
```

## API Reference

### Root Props

- `type`: `"single" | "multiple"` - Allow one or multiple items open
- `collapsible`: `boolean` - Allow collapsing all items (single mode only)
- `defaultValue`: `string | string[]` - Default expanded items

### Accessibility

- Full keyboard navigation with Arrow keys and Enter/Space
- ARIA attributes for screen readers
- Focus management between trigger and content
