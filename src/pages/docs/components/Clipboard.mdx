# Clipboard

One-click copying functionality for text, URLs, and code snippets. Provides user feedback and handles browser permissions gracefully.

## Import

```tsx
import { Clipboard } from "liquidify-react/ark-ui/clipboard";
```

## Basic Usage

```tsx
export default function Example() {
  return (
    <Clipboard.Root value="https://liquidify.dev">
      <Clipboard.Label>Website URL</Clipboard.Label>
      <Clipboard.Control>
        <Clipboard.Input />
        <Clipboard.Trigger>
          <CopyIcon />
        </Clipboard.Trigger>
      </Clipboard.Control>
      <Clipboard.Indicator copied="Copied!" uncoped="Copy">
        Copy URL
      </Clipboard.Indicator>
    </Clipboard.Root>
  );
}
```

## Copy Button Only

```tsx
export default function ButtonExample() {
  return (
    <div style={{ display: "flex", alignItems: "center", gap: "1rem" }}>
      <code>npm install liquidify-react</code>
      <Clipboard.Root value="npm install liquidify-react">
        <Clipboard.Trigger>
          <CopyIcon />
        </Clipboard.Trigger>
      </Clipboard.Root>
    </div>
  );
}
```

## With Custom Feedback

```tsx
export default function FeedbackExample() {
  return (
    <Clipboard.Root value="Custom feedback text">
      <Clipboard.Label>API Key</Clipboard.Label>
      <Clipboard.Control>
        <Clipboard.Input />
        <Clipboard.Trigger>Copy</Clipboard.Trigger>
      </Clipboard.Control>
      <Clipboard.Indicator
        copied={
          <div style={{ color: "green" }}>
            <CheckIcon /> Copied to clipboard!
          </div>
        }
      />
    </Clipboard.Root>
  );
}
```

## Code Block Example

```tsx
export default function CodeExample() {
  const codeSnippet = `import { Button } from "liquidify-react/button"

export default function App() {
  return <Button>Click me</Button>
}`;

  return (
    <div style={{ position: "relative" }}>
      <pre
        style={{
          background: "#f5f5f5",
          padding: "1rem",
          borderRadius: "8px",
          overflow: "auto",
        }}
      >
        <code>{codeSnippet}</code>
      </pre>
      <Clipboard.Root
        value={codeSnippet}
        style={{
          position: "absolute",
          top: "0.5rem",
          right: "0.5rem",
        }}
      >
        <Clipboard.Trigger>
          <CopyIcon />
        </Clipboard.Trigger>
      </Clipboard.Root>
    </div>
  );
}
```

## API Reference

### Root Props

- `value`: `string` - Text to copy to clipboard
- `timeout`: `number` - How long to show copied state (default: 3000ms)

### Trigger Props

- Standard button props for the copy trigger

### Accessibility

- Proper ARIA labels for screen readers
- Keyboard navigation support
- Success/error announcements
- Focus management after copy action
