# Carousel

## Summary
A flexible carousel component for displaying multiple items in a scrollable, navigable format. Perfect for image galleries, product showcases, testimonials, or any content that benefits from horizontal scrolling with navigation controls.

### Import
```ts
import { Carousel } from 'liquidify-react/ark-ui/carousel'
```

## Examples

### Basic Image Carousel
```tsx
import { Carousel } from 'liquidify-react/ark-ui/carousel'

const images = [
  'https://picsum.photos/400/300?random=1',
  'https://picsum.photos/400/300?random=2',
  'https://picsum.photos/400/300?random=3',
  'https://picsum.photos/400/300?random=4',
]

function BasicCarousel() {
  return (
    <Carousel.Root>
      <Carousel.Control>
        <Carousel.ItemGroup>
          {images.map((src, index) => (
            <Carousel.Item key={index}>
              <img
                src={src}
                alt={`Slide ${index + 1}`}
                style={{ width: '100%', height: '200px', objectFit: 'cover' }}
              />
            </Carousel.Item>
          ))}
        </Carousel.ItemGroup>
        <Carousel.PrevTrigger>‹</Carousel.PrevTrigger>
        <Carousel.NextTrigger>›</Carousel.NextTrigger>
      </Carousel.Control>
      <Carousel.IndicatorGroup>
        {images.map((_, index) => (
          <Carousel.Indicator key={index} index={index}>
            {index + 1}
          </Carousel.Indicator>
        ))}
      </Carousel.IndicatorGroup>
    </Carousel.Root>
  )
}
```

### Autoplay Carousel
```tsx
import { Carousel } from 'liquidify-react/ark-ui/carousel'
import { useEffect, useState } from 'react'

function AutoplayCarousel() {
  const [currentIndex, setCurrentIndex] = useState(0)
  const slides = ['Slide 1', 'Slide 2', 'Slide 3', 'Slide 4']

  useEffect(() => {
    const interval = setInterval(() => {
      setCurrentIndex((prev) => (prev + 1) % slides.length)
    }, 3000)
    return () => clearInterval(interval)
  }, [slides.length])

  return (
    <Carousel.Root index={currentIndex} onIndexChange={setCurrentIndex}>
      <Carousel.Control>
        <Carousel.ItemGroup>
          {slides.map((slide, index) => (
            <Carousel.Item key={index}>
              <div style={{
                width: '400px',
                height: '200px',
                backgroundColor: '#e3f2fd',
                display: 'flex',
                alignItems: 'center',
                justifyContent: 'center',
                borderRadius: '8px'
              }}>
                <h2>{slide}</h2>
              </div>
            </Carousel.Item>
          ))}
        </Carousel.ItemGroup>
        <Carousel.PrevTrigger>‹</Carousel.PrevTrigger>
        <Carousel.NextTrigger>›</Carousel.NextTrigger>
      </Carousel.Control>
      <Carousel.IndicatorGroup>
        {slides.map((_, index) => (
          <Carousel.Indicator key={index} index={index} />
        ))}
      </Carousel.IndicatorGroup>
    </Carousel.Root>
  )
}
```

## API Reference

### Carousel.Root
The root container that manages the carousel's state and configuration.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | `0` | Current slide index (controlled) |
| `onIndexChange` | `(index: number) => void` | - | Callback when slide changes |
| `defaultIndex` | `number` | `0` | Initial slide index (uncontrolled) |
| `itemsPerView` | `number \| "auto"` | `1` | Number of items visible at once |
| `spacing` | `string` | - | Spacing between items |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Carousel orientation |
| `loop` | `boolean` | `false` | Enable infinite loop scrolling |
| `dragFree` | `boolean` | `false` | Allow free dragging without snapping |

### Carousel.Control
The main control container that wraps the item group and navigation.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Control content including items and triggers |

### Carousel.ItemGroup
Container for all carousel items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Carousel item elements |

### Carousel.Item
Individual carousel item wrapper.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Item content |

### Carousel.NextTrigger
Button to navigate to the next slide.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger content |
| `asChild` | `boolean` | `false` | Merge props onto child element |

### Carousel.PrevTrigger
Button to navigate to the previous slide.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger content |
| `asChild` | `boolean` | `false` | Merge props onto child element |

### Carousel.IndicatorGroup
Container for slide indicators.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Indicator elements |

### Carousel.Indicator
Individual slide indicator.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | - | **Required.** Slide index this indicator represents |
| `children` | `ReactNode` | - | Indicator content |

## Accessibility

The Carousel component follows WAI-ARIA guidelines for carousel patterns:

- **Keyboard Navigation**: Arrow keys navigate between slides
- **Screen Reader**: Proper ARIA labels and live regions for announcements
- **Focus Management**: Logical tab order and focus indicators
- **Touch Support**: Swipe gestures for mobile devices
- **Reduced Motion**: Respects user's motion preferences

### ARIA Attributes
- `role="region"` with `aria-label` for the carousel container
- `aria-current="true"` for the active slide indicator
- `aria-disabled` for disabled navigation buttons
- `aria-live` for slide change announcements

### Keyboard Shortcuts
- `←/→`: Navigate between slides (horizontal)
- `↑/↓`: Navigate between slides (vertical)
- `Home/End`: Jump to first/last slide
- `Tab`: Navigate to interactive elements

## Best Practices

### Do's
- Use clear navigation controls (arrows, indicators)
- Provide alt text for images in carousels
- Consider autoplay sparingly and provide pause controls
- Use appropriate sizing for touch targets
- Test on various screen sizes and orientations

### Don'ts
- Don't autoplay without user control options
- Don't hide navigation controls on small screens
- Don't use carousels for critical content only
- Don't make slides too small or crowded
- Don't use horizontal scrolling on vertical content

### Performance Tips
- Lazy load images in carousel items
- Use `React.memo` for complex slide content
- Limit visible items with `itemsPerView`
- Optimize images for web delivery
- Consider virtualization for large datasets

### Content Guidelines
- **Images**: Use consistent aspect ratios
- **Text**: Keep content concise and scannable
- **Interactive**: Ensure buttons are properly sized
- **Loading**: Show loading states for dynamic content
