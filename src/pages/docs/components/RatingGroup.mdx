# RatingGroup

A rating component that allows users to rate items using stars or other symbols.

## Installation

```bash
bun add @liquidify/components
```

## Import

```tsx
import { RatingGroup } from "@liquidify/components/rating-group"
```

## Basic Usage

Create a simple star rating component:

```tsx
import { RatingGroup } from "@liquidify/components/rating-group"

export function BasicExample() {
  return (
    <RatingGroup.Root count={5} defaultValue={3}>
      <RatingGroup.Label>Rate this item</RatingGroup.Label>
      <RatingGroup.Control>
        {({ items }) => (
          <>
            {items.map((item) => (
              <RatingGroup.Item key={item} index={item}>
                <RatingGroup.ItemContext>
                  {({ highlighted }) => (
                    <span style={{ color: highlighted ? "#fbbf24" : "#d1d5db" }}>
                      â˜…
                    </span>
                  )}
                </RatingGroup.ItemContext>
              </RatingGroup.Item>
            ))}
          </>
        )}
      </RatingGroup.Control>
    </RatingGroup.Root>
  )
}
```

## Controlled Rating

Control the rating value with state:

```tsx
import { useState } from "react"
import { RatingGroup } from "@liquidify/components/rating-group"

export function ControlledExample() {
  const [rating, setRating] = useState(0)

  return (
    <RatingGroup.Root 
      count={5} 
      value={rating} 
      onValueChange={(details) => setRating(details.value)}
    >
      <RatingGroup.Label>How would you rate our service?</RatingGroup.Label>
      <RatingGroup.Control>
        {({ items }) => (
          <>
            {items.map((item) => (
              <RatingGroup.Item key={item} index={item}>
                <RatingGroup.ItemContext>
                  {({ highlighted }) => (
                    <span style={{ 
                      color: highlighted ? "#f59e0b" : "#e5e7eb",
                      fontSize: "24px",
                      cursor: "pointer"
                    }}>
                      â˜…
                    </span>
                  )}
                </RatingGroup.ItemContext>
              </RatingGroup.Item>
            ))}
          </>
        )}
      </RatingGroup.Control>
      <p>Current rating: {rating}/5</p>
    </RatingGroup.Root>
  )
}
```

## Rating with Feedback

Provide textual feedback based on rating:

```tsx
import { useState } from "react"
import { RatingGroup } from "@liquidify/components/rating-group"

export function FeedbackExample() {
  const [rating, setRating] = useState(0)
  
  const getFeedback = (value) => {
    switch (value) {
      case 1: return "Poor"
      case 2: return "Fair"
      case 3: return "Good"
      case 4: return "Very Good"
      case 5: return "Excellent"
      default: return "No rating"
    }
  }

  return (
    <RatingGroup.Root 
      count={5} 
      value={rating} 
      onValueChange={(details) => setRating(details.value)}
    >
      <RatingGroup.Label>Overall satisfaction</RatingGroup.Label>
      <RatingGroup.Control>
        {({ items }) => (
          <>
            {items.map((item) => (
              <RatingGroup.Item key={item} index={item}>
                <RatingGroup.ItemContext>
                  {({ highlighted }) => (
                    <span style={{ 
                      color: highlighted ? "#10b981" : "#e5e7eb",
                      fontSize: "28px",
                      cursor: "pointer",
                      transition: "color 0.2s"
                    }}>
                      â˜…
                    </span>
                  )}
                </RatingGroup.ItemContext>
              </RatingGroup.Item>
            ))}
          </>
        )}
      </RatingGroup.Control>
      <div style={{ marginTop: "8px", minHeight: "24px" }}>
        {rating > 0 && (
          <p style={{ 
            margin: 0, 
            fontWeight: "600",
            color: rating >= 4 ? "#10b981" : rating >= 3 ? "#f59e0b" : "#ef4444"
          }}>
            {getFeedback(rating)}
          </p>
        )}
      </div>
    </RatingGroup.Root>
  )
}
```

## API Reference

### RatingGroup.Root

The root container for the rating group.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Controlled rating value |
| `defaultValue` | `number` | `0` | Default rating value |
| `onValueChange` | `(details) => void` | - | Callback when rating changes |
| `count` | `number` | `5` | Total number of rating items |
| `allowHalf` | `boolean` | `false` | Whether to allow half ratings |
| `readOnly` | `boolean` | `false` | Whether the rating is read-only |
| `disabled` | `boolean` | `false` | Whether the rating is disabled |

### RatingGroup.Label

The label for the rating group.

### RatingGroup.Control

Container for rating items. Provides render props with `items` array.

### RatingGroup.Item

Individual rating item.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | - | Index of the rating item (required) |

### RatingGroup.ItemContext

Provides context for each rating item with render props.

| Prop | Type | Description |
|------|------|-------------|
| `highlighted` | `boolean` | Whether the item is highlighted |
| `half` | `boolean` | Whether the item shows half rating |

## Accessibility

The RatingGroup component follows WAI-ARIA guidelines:

- Uses `role="radiogroup"` for the container
- Implements proper `aria-label` and `aria-setsize`
- Supports keyboard navigation with arrow keys
- Provides screen reader announcements for rating changes
- Associates labels with `aria-labelledby`
- Implements `aria-valuemin`, `aria-valuemax`, and `aria-valuenow`

## Best Practices

- Provide clear labels indicating what is being rated
- Use appropriate symbols that match your content context
- Consider using half-star ratings for more granular feedback
- Provide textual feedback to supplement visual ratings
- Ensure sufficient contrast between rated and unrated states
- Test with keyboard navigation and screen readers
- Consider mobile touch targets for better usability
- Use read-only mode for displaying existing ratings
