# Pagination

A navigation component for paginated content that provides intuitive page navigation with previous/next controls and page number indicators. Perfect for data tables, search results, content lists, or any scenario requiring navigation through multiple pages of content.

## Installation

```bash
npm install @liquidify/components
```

## Import

```tsx
import { Pagination } from "@liquidify/components/pagination";
```

## Usage

### Basic Pagination

```tsx
import { useState } from "react";
import { Pagination } from "@liquidify/components/pagination";

export default function BasicPagination() {
  const [currentPage, setCurrentPage] = useState(1);
  const totalPages = 10;

  return (
    <div style={{ textAlign: "center", padding: "2rem" }}>
      <h3>Basic Pagination</h3>
      <p>Current page: {currentPage}</p>

      <Pagination.Root
        count={totalPages}
        page={currentPage}
        onPageChange={(details) => setCurrentPage(details.page)}
      >
        <Pagination.PrevTrigger>Previous</Pagination.PrevTrigger>
        <Pagination.Item>1</Pagination.Item>
        <Pagination.Item>2</Pagination.Item>
        <Pagination.Item>3</Pagination.Item>
        <Pagination.Ellipsis />
        <Pagination.Item>10</Pagination.Item>
        <Pagination.NextTrigger>Next</Pagination.NextTrigger>
      </Pagination.Root>
    </div>
  );
}
```

### Data Table with Pagination

```tsx
import { useState, useMemo } from "react";
import { Pagination } from "@liquidify/components/pagination";

export default function DataTablePagination() {
  const [currentPage, setCurrentPage] = useState(1);
  const itemsPerPage = 5;

  // Sample data
  const allData = Array.from({ length: 47 }, (_, i) => ({
    id: i + 1,
    name: `Item ${i + 1}`,
    category: ["Electronics", "Books", "Clothing", "Home", "Sports"][i % 5],
    price: Math.floor(Math.random() * 100) + 10,
    status: ["Active", "Inactive", "Pending"][i % 3]
  }));

  const totalPages = Math.ceil(allData.length / itemsPerPage);

  const currentData = useMemo(() => {
    const startIndex = (currentPage - 1) * itemsPerPage;
    return allData.slice(startIndex, startIndex + itemsPerPage);
  }, [currentPage, allData]);

  const startItem = (currentPage - 1) * itemsPerPage + 1;
  const endItem = Math.min(currentPage * itemsPerPage, allData.length);

  return (
    <div style={{ maxWidth: "800px", margin: "0 auto", padding: "2rem" }}>
      <h3>Data Table with Pagination</h3>

      <div style={{ marginBottom: "1rem", color: "#666" }}>
        Showing {startItem} to {endItem} of {allData.length} entries
      </div>

      <div style={{ border: "1px solid #dee2e6", borderRadius: "8px", overflow: "hidden" }}>
        <table style={{ width: "100%", borderCollapse: "collapse" }}>
          <thead style={{ backgroundColor: "#f8f9fa" }}>
            <tr>
              <th style={{ padding: "1rem", textAlign: "left", borderBottom: "1px solid #dee2e6" }}>ID</th>
              <th style={{ padding: "1rem", textAlign: "left", borderBottom: "1px solid #dee2e6" }}>Name</th>
              <th style={{ padding: "1rem", textAlign: "left", borderBottom: "1px solid #dee2e6" }}>Category</th>
              <th style={{ padding: "1rem", textAlign: "left", borderBottom: "1px solid #dee2e6" }}>Price</th>
              <th style={{ padding: "1rem", textAlign: "left", borderBottom: "1px solid #dee2e6" }}>Status</th>
            </tr>
          </thead>
          <tbody>
            {currentData.map(item => (
              <tr key={item.id} style={{ borderBottom: "1px solid #eee" }}>
                <td style={{ padding: "1rem" }}>{item.id}</td>
                <td style={{ padding: "1rem" }}>{item.name}</td>
                <td style={{ padding: "1rem" }}>{item.category}</td>
                <td style={{ padding: "1rem" }}>${item.price}</td>
                <td style={{ padding: "1rem" }}>
                  <span style={{
                    padding: "0.25rem 0.5rem",
                    borderRadius: "12px",
                    fontSize: "0.8rem",
                    backgroundColor: item.status === "Active" ? "#d4edda" :
                                   item.status === "Inactive" ? "#f8d7da" : "#fff3cd",
                    color: item.status === "Active" ? "#155724" :
                           item.status === "Inactive" ? "#721c24" : "#856404"
                  }}>
                    {item.status}
                  </span>
                </td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>

      <div style={{ marginTop: "2rem", display: "flex", justifyContent: "center" }}>
        <Pagination.Root
          count={totalPages}
          page={currentPage}
          onPageChange={(details) => setCurrentPage(details.page)}
        >
          <Pagination.PrevTrigger>← Previous</Pagination.PrevTrigger>

          {Array.from({ length: Math.min(5, totalPages) }, (_, i) => {
            let pageNum;
            if (totalPages <= 5) {
              pageNum = i + 1;
            } else if (currentPage <= 3) {
              pageNum = i + 1;
            } else if (currentPage >= totalPages - 2) {
              pageNum = totalPages - 4 + i;
            } else {
              pageNum = currentPage - 2 + i;
            }

            return (
              <Pagination.Item key={pageNum} value={pageNum}>
                {pageNum}
              </Pagination.Item>
            );
          })}

          {totalPages > 5 && currentPage < totalPages - 2 && (
            <>
              <Pagination.Ellipsis />
              <Pagination.Item value={totalPages}>{totalPages}</Pagination.Item>
            </>
          )}

          <Pagination.NextTrigger>Next →</Pagination.NextTrigger>
        </Pagination.Root>
      </div>
    </div>
  );
}
```

### Search Results Pagination

```tsx
import { useState, useMemo } from "react";
import { Pagination } from "@liquidify/components/pagination";

export default function SearchResultsPagination() {
  const [currentPage, setCurrentPage] = useState(1);
  const [searchQuery, setSearchQuery] = useState("react");
  const itemsPerPage = 10;

  // Mock search results
  const allResults = useMemo(() => {
    const baseResults = Array.from({ length: 127 }, (_, i) => ({
      id: i + 1,
      title: `${searchQuery} Tutorial ${i + 1}`,
      description: `Learn ${searchQuery} with this comprehensive tutorial covering advanced concepts and best practices.`,
      url: `https://example.com/${searchQuery}-tutorial-${i + 1}`,
      type: ["Article", "Video", "Course", "Documentation"][i % 4],
      date: new Date(Date.now() - Math.random() * 365 * 24 * 60 * 60 * 1000).toLocaleDateString()
    }));

    // Filter based on search query
    return baseResults.filter(result =>
      result.title.toLowerCase().includes(searchQuery.toLowerCase()) ||
      result.description.toLowerCase().includes(searchQuery.toLowerCase())
    );
  }, [searchQuery]);

  const totalPages = Math.ceil(allResults.length / itemsPerPage);

  const currentResults = useMemo(() => {
    const startIndex = (currentPage - 1) * itemsPerPage;
    return allResults.slice(startIndex, startIndex + itemsPerPage);
  }, [currentPage, allResults]);

  const startResult = (currentPage - 1) * itemsPerPage + 1;
  const endResult = Math.min(currentPage * itemsPerPage, allResults.length);

  return (
    <div style={{ maxWidth: "800px", margin: "0 auto", padding: "2rem" }}>
      <h3>Search Results</h3>

      <div style={{ marginBottom: "2rem" }}>
        <div style={{ display: "flex", gap: "1rem", marginBottom: "1rem" }}>
          <input
            type="text"
            value={searchQuery}
            onChange={(e) => {
              setSearchQuery(e.target.value);
              setCurrentPage(1); // Reset to first page on new search
            }}
            placeholder="Search tutorials..."
            style={{
              flex: 1,
              padding: "0.5rem",
              border: "1px solid #ccc",
              borderRadius: "4px"
            }}
          />
          <button
            onClick={() => setCurrentPage(1)}
            style={{
              padding: "0.5rem 1rem",
              backgroundColor: "#007bff",
              color: "white",
              border: "none",
              borderRadius: "4px",
              cursor: "pointer"
            }}
          >
            Search
          </button>
        </div>

        <div style={{ color: "#666", fontSize: "0.9rem" }}>
          Found {allResults.length} results for "{searchQuery}"
          {allResults.length > 0 && (
            <span> • Showing {startResult} to {endResult}</span>
          )}
        </div>
      </div>

      {currentResults.length > 0 ? (
        <div style={{ display: "flex", flexDirection: "column", gap: "1rem", marginBottom: "2rem" }}>
          {currentResults.map(result => (
            <div key={result.id} style={{
              padding: "1rem",
              border: "1px solid #dee2e6",
              borderRadius: "8px",
              backgroundColor: "white"
            }}>
              <div style={{ display: "flex", justifyContent: "space-between", marginBottom: "0.5rem" }}>
                <h4 style={{ margin: 0 }}>
                  <a href={result.url} style={{ color: "#007bff", textDecoration: "none" }}>
                    {result.title}
                  </a>
                </h4>
                <span style={{
                  padding: "0.25rem 0.5rem",
                  backgroundColor: "#e9ecef",
                  borderRadius: "12px",
                  fontSize: "0.8rem"
                }}>
                  {result.type}
                </span>
              </div>
              <p style={{ color: "#666", margin: "0.5rem 0" }}>{result.description}</p>
              <div style={{ fontSize: "0.9rem", color: "#666" }}>
                {result.url} • {result.date}
              </div>
            </div>
          ))}
        </div>
      ) : (
        <div style={{ textAlign: "center", padding: "2rem", color: "#666" }}>
          No results found for "{searchQuery}"
        </div>
      )}

      {totalPages > 1 && (
        <div style={{ display: "flex", justifyContent: "center" }}>
          <Pagination.Root
            count={totalPages}
            page={currentPage}
            onPageChange={(details) => setCurrentPage(details.page)}
          >
            <Pagination.PrevTrigger>Previous</Pagination.PrevTrigger>

            {Array.from({ length: Math.min(7, totalPages) }, (_, i) => {
              let pageNum;
              if (totalPages <= 7) {
                pageNum = i + 1;
              } else if (currentPage <= 4) {
                pageNum = i + 1;
              } else if (currentPage >= totalPages - 3) {
                pageNum = totalPages - 6 + i;
              } else {
                pageNum = currentPage - 3 + i;
              }

              return (
                <Pagination.Item key={pageNum} value={pageNum}>
                  {pageNum}
                </Pagination.Item>
              );
            })}

            {totalPages > 7 && currentPage < totalPages - 3 && (
              <>
                <Pagination.Ellipsis />
                <Pagination.Item value={totalPages}>{totalPages}</Pagination.Item>
              </>
            )}

            <Pagination.NextTrigger>Next</Pagination.NextTrigger>
          </Pagination.Root>
        </div>
      )}
    </div>
  );
}
```

### Infinite Scroll Alternative

```tsx
import { useState, useEffect } from "react";
import { Pagination } from "@liquidify/components/pagination";

export default function PaginationAlternative() {
  const [currentPage, setCurrentPage] = useState(1);
  const [isLoading, setIsLoading] = useState(false);
  const itemsPerPage = 20;

  // Simulate data loading
  const [allData, setAllData] = useState([]);

  useEffect(() => {
    loadPageData(currentPage);
  }, [currentPage]);

  const loadPageData = async (page) => {
    setIsLoading(true);
    // Simulate API call
    await new Promise(resolve => setTimeout(resolve, 1000));

    const newData = Array.from({ length: itemsPerPage }, (_, i) => ({
      id: (page - 1) * itemsPerPage + i + 1,
      title: `Content Item ${(page - 1) * itemsPerPage + i + 1}`,
      description: `This is the description for content item ${(page - 1) * itemsPerPage + i + 1}.`
    }));

    setAllData(prev => page === 1 ? newData : [...prev, ...newData]);
    setIsLoading(false);
  };

  const totalPages = 10; // Assume we know there are 10 pages total

  return (
    <div style={{ maxWidth: "800px", margin: "0 auto", padding: "2rem" }}>
      <h3>Content Feed</h3>

      <div style={{ marginBottom: "2rem" }}>
        {allData.map(item => (
          <div key={item.id} style={{
            padding: "1rem",
            marginBottom: "1rem",
            border: "1px solid #dee2e6",
            borderRadius: "8px",
            backgroundColor: "white"
          }}>
            <h4>{item.title}</h4>
            <p>{item.description}</p>
          </div>
        ))}

        {isLoading && (
          <div style={{ textAlign: "center", padding: "2rem", color: "#666" }}>
            Loading more content...
          </div>
        )}
      </div>

      <div style={{ textAlign: "center" }}>
        <Pagination.Root
          count={totalPages}
          page={currentPage}
          onPageChange={(details) => setCurrentPage(details.page)}
        >
          <Pagination.PrevTrigger disabled={currentPage === 1}>
            Load Previous
          </Pagination.PrevTrigger>

          <span style={{ margin: "0 1rem", color: "#666" }}>
            Page {currentPage} of {totalPages}
          </span>

          <Pagination.NextTrigger
            disabled={currentPage === totalPages || isLoading}
            onClick={() => {
              if (currentPage < totalPages && !isLoading) {
                setCurrentPage(prev => prev + 1);
              }
            }}
          >
            {isLoading ? "Loading..." : "Load More"}
          </Pagination.NextTrigger>
        </Pagination.Root>
      </div>
    </div>
  );
}
```

## API Reference

### Pagination.Root

The root container for the pagination component.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `count` | `number` | - | Total number of pages |
| `page` | `number` | `1` | Current active page |
| `onPageChange` | `(details: PageChangeDetails) => void` | - | Callback fired when page changes |
| `siblingCount` | `number` | `1` | Number of sibling pages to show |
| `boundaryCount` | `number` | `1` | Number of boundary pages to show |
| `children` | `ReactNode` | - | The pagination content |

### Pagination.Item

An individual page number item.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | The page number this item represents |
| `children` | `ReactNode` | - | The item content |

### Pagination.Ellipsis

The ellipsis indicator for skipped page numbers.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The ellipsis content |

### Pagination.PrevTrigger

The previous page navigation trigger.

## Accessibility

The Pagination component includes comprehensive accessibility features:

- **ARIA Roles**: Implements proper navigation and button roles
- **ARIA Labels**: Automatic labeling for screen readers
- **Keyboard Navigation**: Full keyboard support for page navigation
- **Screen Reader Support**: Announces current page and navigation options
- **Focus Management**: Proper focus handling and visual indicators
- **High Contrast**: Compatible with high contrast themes

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Tab` | Navigate between pagination controls |
| `Enter` / `Space` | Activate current page or trigger |
| `Arrow Left/Right` | Navigate between page items |
| `Home` | Go to first page |
| `End` | Go to last page |

## Best Practices

1. **Clear Page Information**: Always show current page and total pages
2. **Consistent Navigation**: Use consistent pagination patterns across your app
3. **Appropriate Page Sizes**: Choose page sizes that balance performance and usability
4. **Loading States**: Show loading indicators when fetching new page data
5. **URL Synchronization**: Keep pagination state in sync with URL parameters
6. **Mobile Friendly**: Ensure pagination works well on mobile devices
7. **Ellipsis Usage**: Use ellipsis appropriately to avoid overwhelming users
8. **Boundary Pages**: Always show first and last page options when appropriate
